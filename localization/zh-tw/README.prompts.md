# 🎯 可重複使用的提示

針對特定開發場景和任務的即用型提示模板，定義了具有特定模式、模型和可用工具集的提示文本。

### 如何使用可重複使用的提示

**安裝方式：**
- 點擊您要使用的提示的 **VS Code** 或 **VS Code Insiders** 安裝按鈕
- 下載 `*.prompt.md` 檔案並手動將其添加到您的提示集合中

**運行/執行方式：**
- 安裝後在 VS Code 聊天中使用 `/prompt-name`
- 從命令面板運行 `Chat: Run Prompt` 命令
- 在 VS Code 中打開提示檔案時點擊運行按鈕
