# 🤖 優秀的 GitHub Copilot 客製化

透過社群貢獻的指令、提示詞和設定，提升您的 GitHub Copilot 體驗。讓 AI 協助始終遵循您的團隊程式碼標準與專案需求。

## 🎯 GitHub Copilot 客製化功能

GitHub Copilot 提供三種主要方式來客製化 AI 回應，並根據您的工作流程、團隊指引和專案需求量身打造協助：

| **📋 [自訂指令](#-自訂指令)** | **🎯 [可重複使用的提示詞](#-可重複使用的提示詞)** | **🧩 [自訂聊天模式](#-自訂聊天模式)** |
| --- | --- | --- |
| 定義程式碼產生、審查、提交訊息等常見任務的指引。描述*如何*執行任務<br><br>**優點：**<br>• 每次聊天請求自動包含<br>• 全倉庫一致性<br>• 多種實作選項 | 建立可重複使用、獨立的提示詞，針對特定任務。描述*要做什麼*，可選擇性加入任務指引<br><br>**優點：**<br>• 省去重複撰寫提示詞<br>• 可跨團隊分享<br>• 支援變數與相依性 | 定義聊天行為、可用工具，以及每次請求的程式碼庫互動模式<br><br>**優點：**<br>• 具情境感知的協助<br>• 工具設定<br>• 角色專屬工作流程 |

> **💡 專家秘訣：** 自訂指令僅影響 Copilot Chat（不影響內嵌程式碼補全）。三種客製化方式可混合使用——一般指引用自訂指令，特定任務用提示詞檔案，聊天模式控制互動情境。
