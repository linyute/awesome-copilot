# 網站探索測試

你的目標是探索網站並找出主要功能。

## 具體指示

1. 使用 Playwright MCP Server 前往指定的網址。若未提供網址，請要求使用者提供。
2. 識別並互動 3-5 個核心功能或使用者流程。
3. 記錄使用者互動、相關 UI 元素（及其定位器）與預期結果。
4. 完成後關閉瀏覽器 context。
5. 提供簡明的探索結果摘要。
6. 根據探索內容提出並產生測試案例。
