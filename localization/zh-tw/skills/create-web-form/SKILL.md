# 建立網頁表單技能

## 概觀

此技能提供建立網頁表單的全面參考資料與最佳實作。內容涵蓋 HTML 語法、UI/UX 設計、表單驗證、伺服器端處理（PHP 與 Python）、資料處理以及網路通訊。

## 目的

藉由提供下列內容，使開發者能夠建構強健、具備無障礙空間且使用者友好的網頁表單：

- HTML 表單語法與結構
- 表單元件的 CSS 樣式設定技術
- 用於表單互動與驗證的 JavaScript
- 無障礙空間最佳實作
- 使用 PHP 與 Python 進行伺服器端表單處理
- 資料庫整合方法
- 網路資料處理與安全
- 效能最佳化

## 用法

建立網頁表單時，請根據您的需求參考對應的檔案：

1. **規劃**： 檢閱 `form-basics.md` 與 `form-controls.md`
2. **結構**： 使用 `html-form-elements.md` 與 `aria-form-role.md`
3. **樣式設定**： 參考 `styling-web-forms.md` 與 `css-styling.md`
4. **互動性**： 套用 `javascript.md` 中的技術
5. **無障礙空間**： 遵循 `accessibility.md` 中的準則
6. **伺服器處理**： 在 PHP 或 Python 參考資料中選擇
7. **資料儲存**： 諮詢資料庫與資料格式參考
8. **最佳化**： 檢閱 `web-performance.md` 與 `security.md`

## 最佳實作

- 務必在用戶端與伺服器端同時進行輸入驗證
- 確保表單對所有使用者皆具備無障礙空間
- 使用語義化 HTML 元件
- 實作正確的錯誤處理與使用者回饋
- 透過 HTTPS 確保表單資料傳輸安全
- 遵循漸進式增強原則
- 在不同的瀏覽器與裝置上測試表單
- 針對效能與使用者體驗進行最佳化
