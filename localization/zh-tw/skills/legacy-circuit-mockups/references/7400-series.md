# 7400 系列邏輯 IC 規格

## 1. 概述

**7400 系列**是一個龐大的**數位邏輯積體電路**家族，最初以 **TTL (電晶體-電晶體邏輯)** 實作，隨後擴展到包含 **CMOS 相容變體**。這些元件提供基礎的建構區塊，例如邏輯閘、正反器、計數器、多工器、解碼器及匯流排收發器。

7400 系列 IC 廣泛用於**復古計算**、**6502/Z80 系統**、**粘合邏輯 (glue logic)** 及教學設計中。

---

## 2. 邏輯家族

| 家族 | 技術 | 註記 |
| ------- | ----------------------- | ----------------------- |
| 74xx | TTL | 原始雙載子 TTL |
| 74LSxx | 低功耗肖特基 TTL | 速度更快，功耗更低 |
| 74HCTxx | CMOS (TTL 位準輸入) | 混合系統的理想選擇 |
| 74HCxx | CMOS | 寬電壓範圍 |
| 74ACTxx | 進階 CMOS TTL 位準 | 速度極快 |

---

## 3. 電氣特性 (典型值)

| 參數 | TTL | CMOS |
| ----------------- | ----------------------- | ------------------- |
| VCC | 5 V | 2-6 V (典型值為 5 V) |
| 輸入高電位 (HIGH) | ≥ 2.0 V | ≥ 0.7xVCC |
| 輸入低電位 (LOW) | ≤ 0.8 V | ≤ 0.3xVCC |
| 扇出 (Fan-out) | ~10 | 極高 |
| 功率消耗 | 較高 | 較低 |

## 8. 定時特性 (一般)

| 參數 | TTL | CMOS |
| ----------------- | ------- | -------- |
| 傳遞延遲 | 5-15 ns | 8-25 ns |
| 最大頻率 | ~25 MHz | ~50+ MHz |

---

## 9. 介面註記

* TTL 輸出可可靠地驅動 TTL 輸入
* CMOS 輸入不可浮動 (float)
* 將 CMOS 與 TTL 訊號連接時，請使用 **74HCT**
* 每個 IC 都需要解耦電容 (0.1 µF)

---

## 10. 典型復古電腦應用

* 位址解碼
* 晶片選擇 (Chip-select) 產生
* 匯流排緩衝
* 時脈閘控與控制
* 狀態機 (State machines)
