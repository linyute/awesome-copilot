# DFRobot FIT0127 LCD 字元顯示器規格

## 1. 概述

**DFRobot FIT0127** 是一個**與 HD44780 相容的字元 LCD 模組**家族，常用於嵌入式系統與愛好者專案。這些顯示器使用點矩陣字元產生器提供英數輸出，並支援 8 位元與 4 位元平行介面。

FIT0127 模組經常與微控制器及 8 位元 CPU 搭配使用，例如 **6502**、**AVR**、**PIC** 與 **Arduino** 平台。

---

## 2. 一般特性

| 特性 | 描述 |
| ----------------- | --------------------------------------------------- |
| 顯示類型 | 字元 LCD (STN，典型為黃綠色背光) |
| 控制器 | HD44780 或相容型號 |
| 介面 | 平行 (4 位元或 8 位元) |
| 字元大小 | 5 x 8 點矩陣 |
| 操作電壓 | 5 V 邏輯位準 (部分變體支援 3.3 V) |
| 背光 | LED (獨立電源腳位) |
| 檢視模式 | 半穿透半反射 (Transflective) |

---

## 3. 顯示變體

FIT0127 型號通常與 **16x2 字元 LCD 模組** 相關聯。

| 變體 | 字元 |
| ------- | ------------------- |
| FIT0127 | 16 欄 x 2 列 |

## 4. 腳位配置

### 4.1 腳位排布 (標準 16 腳位排針)

| 腳位 | 名稱 | 描述 |
| --: | ---- | --------------------- |
|   1 | VSS | 接地 |
|   2 | VDD | +5 V 電源供應 |
|   3 | VO | 對比度調整 |
|   4 | RS | 暫存器選擇 (Register Select) |
|   5 | R/W | 讀取/寫入選擇 |
|   6 | E | 致能 (Enable) |
|   7 | D0 | 資料位元 0 |
|   8 | D1 | 資料位元 1 |
|   9 | D2 | 資料位元 2 |
|  10 | D3 | 資料位元 3 |
|  11 | D4 | 資料位元 4 |
|  12 | D5 | 資料位元 5 |
|  13 | D6 | 資料位元 6 |
|  14 | D7 | 資料位元 7 |
|  15 | A | 背光陽極 (+) |
|  16 | K | 背光陰極 (-) |

| 邏輯電流 | ~1-2 mA |
| 背光電壓 | ~4.2 V |
| 背光電流 | 15-30 mA |

## 6. 介面訊號

### 6.1 RS (暫存器選擇)

| RS | 功能 |
| -- | -------------------- |
| 0 | 指令暫存器 |
| 1 | 資料暫存器 |

### 6.2 R/W

| R/W | 操作 |
| --- | ------------- |
| 0 | 寫入至 LCD |
| 1 | 從 LCD 讀取 |

### 6.3 致能 (Enable, E)

* 資料在 E 的**下降邊緣**被閂鎖
* 每次傳輸都必須讓 E 產生 高 (HIGH)  低 (LOW) 的脈衝

---

## 7. 資料匯流排操作

### 7.1 8 位元模式

* 使用 D0-D7
* 操作速度較快

### 7.2 4 位元模式

* 僅使用 D4-D7
* 資料以兩個四位元 (nibbles) 傳輸 (高位在前)
* 節省 I/O 腳位

---

## 8. 內部記憶體映射表

### 8.1 DDRAM (顯示資料 RAM)

| 位址 | 顯示位置 |
| --------: | ---------------- |
| 0x00-0x0F | 第 1 行，第 1-16 欄 |
| 0x40-0x4F | 第 2 行，第 1-16 欄 |

### 8.2 CGRAM (字元產生器 RAM)

* 支援最多 **8 個自訂字元**
* 每個字元使用 8 個位元組

---

## 9. 指令集 (摘要)

| 指令 | 描述 |
| ----------- | ---------------------- |
| 0x01 | 清除顯示 |
| 0x02 | 歸位 (Return home) |
| 0x04-0x07 | 進入模式設定 |
| 0x08-0x0F | 顯示開啟/關閉控制 |
| 0x10-0x1F | 游標/顯示位移 |
| 0x20-0x3F | 功能設定 |
| 0x40-0x7F | 設定 CGRAM 位址 |
| 0x80-0xFF | 設定 DDRAM 位址 |

---

## 10. 初始化序列 (4 位元模式)

```text
VDD 上升後等待 >15 ms
功能設定：0x33
功能設定：0x32
功能設定：0x28 (4 位元，2 行)
顯示開啟/關閉：0x0C
進入模式：0x06
清除顯示：0x01
```

## 11. 定時特性 (典型值)

| 操作 | 時間 |
| ------------------ | -------- |
| 致能脈衝寬度 | ≥ 450 ns |
| 指令執行 | ~37 µs |
| 清除 / 歸位 | ~1.52 ms |

---

## 12. 典型系統整合 (以 6502 為例)

```text
RS   VIA 輸出
E    VIA 輸出
D4-D7  VIA Port B
R/W  接地 (僅限寫入)
```

---

## 13. 對比度與背光控制

* 對比度透過 VO 腳位上的可變電阻調整
* 背光可能需要串聯電阻
* 支援透過外部控制進行 PWM 調光

---

## 14. 絕對最大額定值 (摘要)

| 輸入電壓 | -0.3 V 至 VDD + 0.3 V |
| 操作溫度 | -20 °C 至 +70 °C |
