# 免焊麵包板

這是一份關於**常見免焊麵包板**的實用 Markdown 規格與參考，旨在用於電子原型設計、教學及愛好者開發。

---

## 1. 概述

**免焊麵包板**是一種可重複使用的原型設計平台，允許在不焊接的情況下互連電子元件。連接是透過內部的彈簧夾實現的。

### 典型用途

* 快速電路原型設計
* 教學實驗室
* 邏輯與微控制器實驗
* 低功耗類比與數位電路

### 不適用於

* 高電流 (>1A)
* 高電壓 (>36V)
* 射頻 (RF) / 高頻設計
* 易受震動或永久性安裝

---

## 2. 物理結構

### 材料

* ABS 或聚苯乙烯機身
* 磷青銅或鍍鎳彈簧接點
* 粘性背膠 (選配)

### 標準孔距

* **2.54 mm (0.1 in)** - 與 DIP IC 及標準排針相容

### 接點特性

| 參數 | 典型值 |
| ---------------- | ------------- |
| 接點電阻 | 10-50 mΩ |
| 插拔次數 | ~5,000 |
| 導線線徑 | 20-28 AWG |

---

## 3. 內部電氣連接

### 終端條 (主要區域)

* 每列由 **5 個互連孔**組成
* 水平連通性
* 中心溝槽隔離左右兩半

```
A B C D E | F G H I J
──────────┼──────────
  已連接   |   已連接
```

### 電源軌

* 每側的垂直匯流排
* 通常在**中間斷開**（不一定是連續的）
* 通常標示為 **紅色 (+)** 與 **藍色 (-)**

```
+  +  +  +  +   (可能會斷開)
-  -  -  -  -
```

> ⚠️ 務必使用三用電表驗證連通性

| --------- | ---------- | ------------------------ |
| 小型 (Mini) | 170 | 小型測試電路 |

| 模組化 | 變動 | 可擴充系統 |

---

## 5. 元件相容性

### 相容元件

* DIP IC (300 mil, 600 mil)
* 軸向電阻與二極體
* LED
* 輕觸開關
* 跳線
* 排針

### 有問題的元件

| 元件 | 問題 |
| -------------------- | -------------------------- |
| TO-220 | 太寬 / 使接點受壓 |
| SMD | 需要轉接板 |
| 大型電解電容 | 機械應變 |
| 高功率電阻 | 發熱 |

---

## 6. 電氣特性

### 典型限制

| 參數 | 建議最大值 |
| ------------------- | ------------------ |
| 電壓 | 30-36 V |
| 每個接點電流 | 500-1000 mA |
| 頻率 | <1 MHz (實務上) |

### 寄生效應 (近似值)

| 類型 | 數值 |
| ----------- | ------------------- |
| 電容 | 每個節點 1-5 pF |
| 電感 | 每個跳線 10-20 nH |

---

## 7. 最佳實踐

### 電源分配

* 將**接地與 Vcc** 連接到兩側
* 如果需要，橋接斷開的電源軌
* 為每個 IC 使用 **0.1µF 陶瓷電容**進行解耦

### 接線

* 保持接線短促且整潔
* 使用顏色編碼：

  * 紅色：Vcc
  * 黑色/藍色：GND
  * 黃色/白色：訊號

### IC 放置

* 將 DIP IC **跨接在中心溝槽**上
* 避免強行插入腳位

---

## 8. 常見錯誤

| 錯誤 | 結果 |
| ----------------------------- | ------------------ |
| 假設導軌是連續的 | 電源流失 |
| 過長的跳線 | 雜訊、不穩定 |
| 未使用解耦電容 | 行為異常 |
| 超過電流限制 | 接點熔化 |

---

## 9. 測試與偵錯

### 連通性檢查

* 使用三用電表驗證導軌與行列

### 訊號完整性提示

* 避免將麵包板用於：

  * 高速時脈
  * ADC 精密電路
