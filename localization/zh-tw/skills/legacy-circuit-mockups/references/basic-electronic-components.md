# 基礎電子元件

此文件涵蓋了四種基礎電子元件：電阻、電容、按鈕與開關，包含常見數值、公式及類型。

---

## 1. 電阻 (R)

電阻限制電流流動，並將能量以熱的形式散發。

### 常見公式

* **歐姆定律：** $V = I \times R$ (電壓 = 電流 $\times$ 電阻)
* **串聯電阻：** $R_{total} = R_1 + R_2 + ... + R_n$
* **並聯電阻：** $\frac{1}{R_{total}} = \frac{1}{R_1} + \frac{1}{R_2} + ... + \frac{1}{R_n}$

### 常見數值 (E12/E24 系列 - 5% 誤差)

數值乘以上 10 的冪次方 (例如 10Ω, 100Ω, 1kΩ, 10kΩ, 100kΩ, 1MΩ)：
**1.0, 1.1, 1.2, 1.3, 1.5, 1.6, 1.8, 2.0, 2.2, 2.4, 2.7, 3.0, 3.3, 3.6, 3.9, 4.3, 4.7, 5.1, 5.6, 6.2, 6.8, 7.5, 8.2, 9.1**

## 2. 電容 (C)

電容將電荷儲存在電場中。

### 常見公式

* **電容量：** $C = \frac{Q}{V}$ (電荷 / 電壓)
* **電流：** $i = C \frac{dV}{dt}$
* **並聯電容：** $C_{total} = C_1 + C_2 + ... + C_n$
* **串聯電容：** $\frac{1}{C_{total}} = \frac{1}{C_1} + \frac{1}{C_2} + ... + \frac{1}{C_n}$
* **儲存能量：** $W = \frac{1}{2} C V^2$

### 常見數值

* **陶瓷電容 (picoFarads/nanoFarads)：** 10pF, 22pF, 100pF, 1nF, 10nF, 100nF (通常標示為 "104")
* **電解電容 (microFarads)：** 1µF, 10µF, 22µF, 47µF, 100µF, 220µF, 470µF, 1000µF

---

## 3. 按鈕 (瞬間按鈕)

一種瞬間開關，僅在按下時接通電路。

### 常見類型

* **常開 (Normally Open, NO)：** 電路處於斷路狀態（關閉），直到被按下。
* **常閉 (Normally Closed, NC)：** 電路處於通路狀態（開啟），直到被按下。
* **輕觸開關 (Tactile Switch)：** 用於 PCB 安裝的小型標準按鈕。

---

## 4. 開關 (SW)

一種機電裝置，可斷開或連接電路，並在切換前保持在原位。

### 常見類型

* **SPST：** 單刀單擲 (開/關)
* **SPDT：** 單刀雙擲 (在兩條路徑之間切換)
* **DPST/DPDT：** 雙刀變體 (同時控制兩個獨立電路)
* **DIP 開關：** 用於電路板的小型開關陣列。

| 元件 | 符號 (代號) | 關鍵公式 | 常見用途 |
| :--- | :--- | :--- | :--- |
