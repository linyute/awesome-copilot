# T48 通用 EEPROM / Flash 燒錄器規格

## 1. 概述

**T48 通用燒錄器**（亦以 **TL866-3G / TL866II Plus 繼任者** 之名銷售）是一款透過 USB 連接的裝置，用於**讀取、程式化與驗證**各式各樣的 **EEPROM、Flash、EPROM、GAL 與微控制器元件**。它常用於**復古計算**、**韌體開發**與**電子維修**工作流。

T48 是 TL866 系列的現代化替代品，提供擴展的元件支援與改進的軟體相容性。

---

## 2. 一般特性

| 特性 | 描述 |
| ------------------- | ---------------------------------- |
| 燒錄器類型 | 通用元件燒錄器 |
| 介面 | USB 2.0 |
| 插槽 | ZIF-48 |
| 支援元件 | EEPROM, Flash, EPROM, GAL, MCU |
| 燒錄電壓 | 內部產生 |
| 主機作業系統 | Windows, Linux (開源工具) |
| 驗證方式 | 寫入後讀取、總和檢查碼 (checksum) |

---

## 3. 物理描述

### 3.1 ZIF 插槽

* **48 腳位零插拔力 (ZIF)** 插槽
* 直接支援 DIP 封裝
* 透過轉接板支援 PLCC、SOP、TSOP

### 3.2 指示燈與控制項

| 項目 | 描述 |
| ------------- | --------------------------- |
| 狀態 LED | 電源 / 活動指示 |
| ZIF 拉桿 | 將 IC 鎖定於插槽中 |
| USB 接頭 | 主機連接與電源 |

---

## 4. 支援的元件類別

### 4.1 記憶體元件

* 27xxx EPROM
* 28xxx EEPROM (例如 AT28C256)
* 29xxx Flash 記憶體
* 序列式 EEPROM (需使用轉接板)

### 4.2 可程式化邏輯

* GAL16V8, GAL22V10 (讀取/寫入/驗證)

### 4.3 微控制器 (有限支援)

* 部分 PIC、AVR 與 8051 家族元件
* 燒錄支援取決於電壓與腳位排布

---

## 5. 電氣能力

| 參數 | 描述 |
| -------------- | ---------------------------------------- |
| VCC 範圍 | 1.8 V - 6.5 V (視元件而定) |
| VPP | 內部產生，視元件而定 |
| I/O 保護 | 具備過電流與短路保護 |

---

## 6. 燒錄操作

### 6.1 常見操作

* 元件識別
* 空白檢查 (Blank check)
* 讀取
* 程式化 (寫入)
* 驗證
* 抹除 (Flash/EPROM)

### 6.2 驗證模式

* 逐位元組比較
* 總和檢查碼 (Checksum) / CRC 驗證

---

## 7. 軟體支援

### 7.1 官方軟體

* 基於 Windows 的圖形化介面 (GUI)
* 具備腳位映射的元件資料庫
* 自動電壓與時序控制

### 7.2 開源工具

| 工具 | 註記 |
| --------- | -------------------------------- |
| `minipro` | Linux / macOS / Windows CLI 工具 |
| libusb | USB 通訊後端 |

範例：

```bash
minipro -p AT28C256 -w rom.bin
```

---

## 8. 元件插入與腳位映射

* 元件方向由 **第 1 腳位標記** 指示
* 許多元件在 ZIF 插槽中使用 **左下角對齊**
* 軟體會顯示正確的插入示意圖

---

## 9. 典型工作流 (以 EEPROM 為例)

1. 選擇元件類型 (例如 AT28C256)
2. 將晶片插入 ZIF 插槽
3. 執行空白檢查
4. 載入二進位映像檔
5. 程式化元件
6. 驗證寫入內容

---

## 10. 電源與安全註記

* 燒錄器完全透過 USB 供電
* 燒錄期間請勿插入或移除 IC
* 對於非 DIP 封裝，請使用轉接板

---

## 11. 限制

* 並非支援所有微控制器
* 高壓 EPROM 可能需要特定的轉接板
* 不適用於在線燒錄 (In-circuit programming, ISP)

---

## 12. 常見案例

* 為 6502 SBC 燒錄 EEPROM
* 為復古系統快閃 (Flashing) ROM 映像
* 讀取與備份舊式 EPROM
* GAL 邏輯開發

---

## 13. 與 TL866II Plus 之比較

| 特性 | TL866II Plus | T48 |

| 元件支援 | 良好 | 已擴展 |
| 作業系統支援 | Windows | Windows / Linux |
| 開源工具 | 有限 | 優異 |
