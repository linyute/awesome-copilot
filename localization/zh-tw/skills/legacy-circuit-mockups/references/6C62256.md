# AS6C62256 32K x 8 低功耗 CMOS SRAM 規格

## 1. 概述

**AS6C62256** 是一款 **256 Kbit (32K x 8)** 的低功耗 CMOS 靜態隨機存取記憶體 (SRAM)，由 Alliance Memory 製造（亦有其他來源）。它與常見的 28 腳位 SRAM 腳位完全相容，並廣泛用於 **6502**、**Z80** 及其他 8 位元微處理器系統，作為讀/寫記憶體使用。

此元件提供快速的存取時間、簡單的控制邏輯以及極低的待機功耗。

---

## 2. 一般特性

| 特性 | 描述 |
| ----------------- | --------------------------------- |
| 記憶體大小 | 256 Kbits (32 KB) |
| 組織方式 | 32,768 x 8 位元 |
| 資料匯流排 | 8 位元 |
| 位址匯流排 | 15 位元 (A0-A14) |
| 技術 | CMOS SRAM |
| 存取時間 | 55 ns / 70 ns (依型號而定) |
| 操作電壓 | 5 V ± 10% |
| 待機電流 | < 1 µA (典型值) |
| 封裝類型 | DIP-28, SOJ-28, TSOP-28 |

---

## 3. 腳位配置 (邏輯)

### 3.1 位址腳位 (A0-A14)

* 選擇 32,768 個記憶體位置中的一個
* 位址在讀取或寫入週期期間必須保持穩定

### 3.2 資料腳位 (I/O0-I/O7)

* 雙向三態資料匯流排
* 未致能時處於高阻抗狀態

### 3.3 控制腳位

| 腳位 | 描述 |
| --- | -------------------------- |
| CE# | 晶片致能 (Chip Enable) (低電位有效) |
| OE# | 輸出致能 (Output Enable) (低電位有效) |
| WE# | 寫入致能 (Write Enable) (低電位有效) |
| VCC | +5 V 電源供應 |
| GND | 接地 |

---

## 4. 記憶體組織

* 從 `$0000` 到 `$7FFF` 的線性位址空間
* 每個位址對應一個位元組 (8 位元)

```text
位址範圍：0000h - 7FFFh
資料寬度：8 位元
```

---

## 5. 讀取操作

### 5.1 讀取週期條件

| 訊號 | 狀態 |
| ------ | ----- |
| CE# | 低 (LOW) |
| OE# | 低 (LOW) |
| WE# | 高 (HIGH) |

* 資料在存取時間 (tAA) 後變為有效
* 當 CE# 與 OE# 為低電位 (LOW) 時，輸出保持有效
* 當 CE# 或 OE# 為高電位 (HIGH) 時，輸出為高阻抗

---

## 6. 寫入操作

### 6.1 寫入週期條件

| 訊號 | 狀態 |
| ------ | ----------- |
| CE# | 低 (LOW) |
| OE# | 高或低 |
| WE# | 低脈衝 (LOW pulse) |

* 資料在 WE# 或 CE# 的上升邊緣被寫入
* 位址與資料在寫入視窗期間必須保持穩定
* 無內部寫入延遲（真實的 SRAM 行為）

---

## 7. 定時註記 (摘要)

| 參數 | 典型值 |
| -------------------- | -------- |
| 位址存取 (tAA) | 55-70 ns |
| CE# 存取 (tACE) | 55-70 ns |
| OE# 存取 (tOE) | 25-35 ns |
| 寫入週期時間 | ≥ 55 ns |

---

## 8. 電源模式

### 8.1 主動模式

* CE# = 低 (LOW)
* 正常的讀取/寫入操作

### 8.2 待機模式

* CE# = 高 (HIGH)
* 資料保留
* 極低的功耗

---

## 9. 重設與啟動電源行為

* 無需重設腳位
* 啟動電源時資料為未定義
* 僅在存在 VCC 時保留記憶體內容

---

## 10. 典型系統整合 (以 6502 為例)

```text
映射於：$0000 - $7FFF
CE#  經編碼的位址
OE#  R/W?
WE#  反向 R/W?
```
