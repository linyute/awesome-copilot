# AT28C256 256K (32K x 8) 平行 EEPROM 規格

## 1. 概述

**AT28C256** 是由 Atmel（現為 Microchip）製造的一款非揮發性、電氣可抹除且可程式化的唯讀記憶體 (EEPROM)。它提供 **256 Kbits** 的儲存空間，組織為 **32,768 x 8 位元**，常用於 8 位元微處理器系統，例如基於 **6502**、**Z80** 及類似 CPU 的系統。

該元件支援位元組級寫入操作、快速讀取存取以及軟體控制的資料保護。

---

## 2. 一般特性

| 特性 | 描述 |
| -------------- | ------------------------------ |
| 記憶體大小 | 256 Kbits (32 KB) |
| 組織方式 | 32,768 x 8 位元 |
| 資料匯流排 | 8 位元 |
| 位址匯流排 | 15 位元 (A0-A14) |
| 技術 | EEPROM |
| 耐久性 | ≥ 100,000 次寫入週期 |
| 資料保留 | ≥ 10 年 |
| 存取時間 | 150-250 ns (依型號而定) |
| 封裝類型 | DIP-28, PLCC-32, TSOP |

---

## 3. 腳位配置 (邏輯)

### 3.1 位址腳位 (A0-A14)

* 選擇 32,768 個記憶體位置中的一個

### 3.2 資料腳位 (I/O0-I/O7)

* 雙向三態資料匯流排
* 在讀取週期期間輸出有效

### 3.3 控制腳位

| 腳位 | 描述 |
| --- | -------------------------- |
| CE | 晶片致能 (Chip Enable) (低電位有效) |
| OE | 輸出致能 (Output Enable) (低電位有效) |
| WE | 寫入致能 (Write Enable) (低電位有效) |
| VCC | +5 V 電源供應 |
| GND | 接地 |

---

## 4. 記憶體組織

* 從 `$0000` 到 `$7FFF` 的線性位址空間
* 每個位址對應一個 8 位元的位元組

```text
位址範圍：0000h - 7FFFh
資料寬度：8 位元
```

---

## 5. 讀取操作

### 5.1 讀取週期條件

| 訊號 | 狀態 |
| ------ | ----- |
| CE | 低 (LOW) |
| OE | 低 (LOW) |
| WE | 高 (HIGH) |

* 資料在存取時間後出現在 I/O 腳位上
* 當 CE 與 OE 處於有效狀態時，輸出保持有效
* 當 CE 或 OE 為高電位 (HIGH) 時，輸出為高阻抗 (high-impedance)

---

## 6. 寫入操作

### 6.1 位元組寫入週期

| 訊號 | 狀態 |
| ------ | --------- |
| CE | 低 (LOW) |
| OE | 高 (HIGH) |
| WE | 低脈衝 (LOW pulse) |

* 在 WE 低脈衝期間，位址與資料必須保持穩定
* 內部寫入週期時間 ≈ 10 ms (最大值)
* 元件會自動處理「先抹除後寫入」

---

## 7. 軟體資料保護 (SDP)

AT28C256 包含選配的 **軟體資料保護**，以防止意外寫入。

### 7.1 SDP 啟動程序

```text
將 $AA 寫入位址 $5555
將 $55 寫入位址 $2AAA
將 $A0 寫入位址 $5555
```

### 7.2 SDP 停用程序

```text
將 $AA 寫入位址 $5555
將 $55 寫入位址 $2AAA
將 $80 寫入位址 $5555
將 $AA 寫入位址 $5555
將 $55 寫入位址 $2AAA
將 $20 寫入位址 $5555
```

---

## 8. 寫入週期定時註記

* 寫入操作採內部定時，無需外部輪詢 (polling)
* 在寫入週期期間，讀取操作會回傳未定義的資料
* 忙碌時，元件會忽略額外的寫入嘗試

---

## 9. 資料輪詢 (選配)

* 在寫入期間可監測 I/O7
* 當 I/O7 與寫入的資料相符時，寫入即完成

---

## 10. 重設與電源行為

* 無明確的重設腳位
* 在啟動電源與關閉電源期間禁止寫入
* 輸出預設為高阻抗，直到 CE 與 OE 被觸發

---

## 11. 典型系統整合 (以 6502 為例)

```text
位址範圍：$8000 - $FFFF
A15 用作晶片選擇
OE  R/W?
WE  反向 R/W?
```

## 12. 絕對最大額定值 (摘要)

| 參數 | 額定值 |
| ------------- | --------------------- |
| VCC | -0.6 V 至 +6.25 V |
| 輸入電壓 | -0.6 V 至 VCC + 0.6 V |
| 儲存溫度 | -65 °C 至 +150 °C |
