# 常見麵包板電子元件

這是一份關於**基於麵包板的電子產品**實用參考，涵蓋常見元件、公式、表格、腳位及學習參考資料。適用於從初學者到中階硬體建置者。

---

## 1. 電阻

### 描述

電阻限制電流、分壓並設定偏壓點。

### 常見類型

| 類型 | 註記 | 典型用途 |
| ------------- | ----------- | ------------------ |
| 碳膜電阻 | 便宜，±5% | 一般用途 |
| 金屬膜電阻 | 穩定，±1% | 精密電路 |
| 繞線電阻 | 高功率 | 功率耗散 |
| 可變電阻 | 可調整 | 音量、調諧 |

### 標準數值 (E12 系列)

`10, 12, 15, 18, 22, 27, 33, 39, 47, 56, 68, 82 x 10^n`

### 顏色代碼

| 色環 | 意義 |
| ---- | ------------ |
| 1 | 第一位數字 |
| 2 | 第二位數字 |
| 3 | 倍數 |
| 4 | 容差 (誤差) |

### 額定功率

| 額定值 | 典型尺寸 |
| ------ | ------------------- |
| 1/8 W | 小型 |
| 1/4 W | 麵包板標準 |
| 1/2 W | 大型 |

### 公式

* **歐姆定律：** `V = I x R`
* **功率：** `P = V × I = I² × R = V² / R`

### 參考

* [https://en.wikipedia.org/wiki/Resistor](https://en.wikipedia.org/wiki/Resistor)

## 2. 電容

### 描述

電容儲存電能，並在通過交流電 (AC) 的同時阻隔直流電 (DC)。

### 常見類型

| 類型 | 具極性 | 典型用途 |
| ------------ | --------- | ------------------ |
| 陶瓷電容 | 否 | 解耦、旁路 |
| 電解電容 | 是 | 大容量濾波 |
| 鉭質電容 | 是 | 緊湊型濾波 |
| 薄膜電容 | 否 | 訊號耦合 |

### 常見數值

| 範圍 | 範例 |
| ----- | ---------------- |
| pF | 10pF, 100pF |
| nF | 100nF (0.1µF) |
| µF | 1µF, 10µF, 100µF |

### 耐壓規則

> 電容額定電壓 ≥ **2 倍電路電壓**

### 公式

* **容抗：** `Xc = 1 / (2πfC)`
* **儲存能量：** `E = ½ C V²`

### 參考

* [https://en.wikipedia.org/wiki/Capacitor](https://en.wikipedia.org/wiki/Capacitor)

## 3. 電感

### 描述

電感將能量儲存在磁場中，並抵抗電流變動。

### 常見類型

| 類型 | 用途 |
| ------------ | ----------------- |
| 空心電感 | 射頻 (RF) 電路 |
| 鐵氧體磁芯電感 | 電源濾波 |
| 扼流圈 (Choke) | 雜訊抑制 |

### 公式

* **感抗：** `XL = 2πfL`

### 參考

* [https://en.wikipedia.org/wiki/Inductor](https://en.wikipedia.org/wiki/Inductor)

---

## 4. 二極體

### 描述

二極體僅允許電流以單一方向流動。

### 常見類型

| 類型 | 典型零件 | 用途 |
| --------- | ------------ | --------------------- |
| 整流器 | 1N4007 | 電源 |
| 訊號二極體 | 1N4148 | 邏輯、快速切換 |
| 齊納二極體 | 1N4733A | 電壓調節 |
| 肖特基二極體 | 1N5819 | 低壓降 |

### 關鍵參數

| 參數 | 典型值 |
| --------------- | -------------------------- |
| 順向電壓 | 0.7V (矽), 0.3V (肖特基) |
| 反向電壓 | 依元件而定 |

### 參考

* [https://en.wikipedia.org/wiki/Diode](https://en.wikipedia.org/wiki/Diode)

## 5. LED (發光二極體)

### 描述

LED 在順向偏壓時會發光。

### 典型順向電壓

| 顏色 | Vf |
| ----- | -------- |
| 紅色 | 1.8-2.2V |
| 綠色 | 2.0-3.0V |
| 藍色 | 3.0-3.3V |

### 限流電阻

`R = (V_supply - V_LED) / I_LED`

### 參考

* [https://en.wikipedia.org/wiki/Light-emitting_diode](https://en.wikipedia.org/wiki/Light-emitting_diode)

## 6. 電晶體

### BJT (雙載子接面電晶體)

| 類型 | 範例 | 用途 |
| ---- | ------- | ------------------------ |
| NPN | 2N3904 | 切換、放大 |
| PNP | 2N3906 | 高端切換 |

**關鍵公式：**

* `Ic ≈ β × Ib`

| 類型 | 範例 | 用途 |
| --------- | ------- | ------------------------ |

| P 通道 | IRF9540 | 高端控制 |

## 7. 積體電路 (DIP)

### 常見邏輯家族

| 家族 | 電壓 | 註記 |
| ----------- | ------- | ------------------- |
| TTL (74LS) | 5V | 舊式 (Legacy) |
| CMOS (74HC) | 2-6V | 適合麵包板 |

### 解耦規則

> 每個 IC 都要在 Vcc 與 GND 之間放置 **0.1µF 陶瓷電容**

## 9. 麵包板

### 描述

具有內部匯流排連接的免焊原型設計板。

### 內部接線

* 行：水平連接 (5 個孔)
* 軌：垂直連接 (電源)

### 限制

* 不適用於高頻或高電流電路

### 參考

* [https://en.wikipedia.org/wiki/Breadboard](https://en.wikipedia.org/wiki/Breadboard)
