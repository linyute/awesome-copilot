# 視覺檢查清單 (Visual Inspection Checklist)

本文件是一份全面的清單，列出了網頁設計視覺檢查期間需要驗證的項目。

---

## 1. 版面配置驗證 (Layout Verification)

### 結構完整性

- [ ] 頁首 (Header) 已正確固定/定位在螢幕頂部
- [ ] 頁尾 (Footer) 定位在螢幕底部或內容末尾
- [ ] 主要內容區域置中對齊且寬度適中
- [ ] 側邊欄 (如有) 定位正確
- [ ] 導覽列顯示在預期位置

### 溢出

- [ ] 未意外顯示水平捲軸
- [ ] 內容未超出父元件
- [ ] 圖片符合父容器大小
- [ ] 表格未超過容器寬度
- [ ] 程式碼區塊可正確換行或捲動

### 對齊

- [ ] Grid 項目均勻分佈
- [ ] Flex 項目對齊正確
- [ ] 文字對齊 (左/中/右) 一致
- [ ] 圖示與文字垂直對齊
- [ ] 表單標籤與輸入欄位定位正確

---

## 2. 字體排版驗證 (Typography Verification)

### 可讀性

- [ ] 本文文字字體大小足夠 (建議至少 16px)
- [ ] 行高適中 (建議 1.5-1.8)
- [ ] 每行字數適中 (建議 40-80 個字元)
- [ ] 段落之間間距足夠
- [ ] 標題層級清晰

### 文字處理

- [ ] 長單字可正確換行
- [ ] URL 與程式碼處理妥當
- [ ] 未發生文字截斷
- [ ] 刪節號 (...) 顯示正確
- [ ] 語言特定的換行規則運作正確

### 字體

- [ ] 網頁字體 (Web fonts) 載入正確
- [ ] 遞補字體 (Fallback fonts) 合適
- [ ] 字重 (Font weights) 符合預期
- [ ] 特殊字元與表情符號顯示正確

---

## 3. 顏色與對比度驗證 (Color & Contrast Verification)

### 無障礙性 (WCAG 標準)

- [ ] 本文文字：對比率 4.5:1 或更高 (AA)
- [ ] 大型文字 (18px+ 粗體或 24px+)：3:1 或更高
- [ ] 互動式元件邊框：3:1 或更高
- [ ] 焦點指示器：與背景有足夠對比度

### 顏色一致性

- [ ] 品牌顏色統一
- [ ] 連結顏色一致
- [ ] 錯誤狀態的紅色統一
- [ ] 成功狀態的綠色統一
- [ ] 懸停/啟動 (Hover/Active) 狀態顏色合適

### 色覺多樣性

- [ ] 透過形狀和文字傳達資訊，而非僅靠顏色
- [ ] 圖表考慮了色覺多樣性
- [ ] 錯誤訊息不單獨依賴顏色

---

## 4. 回應式驗證 (Responsive Verification)

### 行動裝置 (~640px)

- [ ] 內容符合螢幕寬度
- [ ] 觸控目標為 44x44px 或更大
- [ ] 文字大小可輕易閱讀
- [ ] 未出現水平捲動
- [ ] 導覽列適合行動裝置 (漢堡選單等)
- [ ] 表單輸入易於使用

### 平板電腦 (641px~1024px)

- [ ] 版面針對平板電腦進行優化
- [ ] 雙欄版面配置顯示正確
- [ ] 圖片尺寸合適
- [ ] 側邊欄顯示/隱藏切換正常

### 桌上型電腦 (1025px~)

- [ ] 已設定最大寬度，且在超大螢幕上不會破裂
- [ ] 間距足夠
- [ ] 多欄版面配置運作正常
- [ ] 已實作懸停 (Hover) 狀態

### 中斷點轉換

- [ ] 螢幕尺寸變更時，版面轉換流暢
- [ ] 在中間尺寸時版面不會破裂
- [ ] 無內容消失或重複

---

## 5. 互動式元件驗證 (Interactive Element Verification)

### 按鈕

- [ ] 預設狀態清晰
- [ ] 具備懸停 (Hover) 狀態 (桌上型電腦)
- [ ] 焦點 (Focus) 狀態視覺清晰
- [ ] 具備啟動 (Active，即按下) 狀態
- [ ] 停用 (Disabled) 狀態可輕易區分
- [ ] 載入狀態 (如適用)

### 連結

- [ ] 連結在視覺上可識別
- [ ] 已瀏覽過的連結可區分 (如需要)
- [ ] 具備懸停狀態
- [ ] 焦點狀態清晰

### 表單元件

- [ ] 輸入欄位邊界清晰
- [ ] 預留位置 (Placeholder) 文字對比度合適
- [ ] 獲得焦點時有視覺回饋
- [ ] 錯誤狀態顯示
- [ ] 必填欄位標示
- [ ] 下拉選單運作正常

---

## 6. 圖片與媒體驗證 (Images & Media Verification)

### 圖片

- [ ] 圖片以合適尺寸顯示
- [ ] 維持長寬比
- [ ] 支援高解析度顯示 (@2x)
- [ ] 圖片載入失敗時的顯示方式
- [ ] 延遲載入 (Lazy loading) 行為正常

### 影片與內嵌內容

- [ ] 影片符合容器大小
- [ ] 維持長寬比
- [ ] 內嵌內容具備回應式能力
- [ ] iframe 未溢出

---

## 7. 無障礙性驗證 (Accessibility Verification)

### 鍵盤導覽

- [ ] 所有互動式元件皆可透過 Tab 鍵存取
- [ ] 焦點順序符合邏輯
- [ ] 焦點陷阱 (Focus traps) 設定合適 (如互動視窗)
- [ ] 具備「跳至主要內容」連結

### 螢幕閱讀器支援

- [ ] 圖片具備 alt 文字
- [ ] 表單具備標籤 (Labels)
- [ ] ARIA 標籤設定合適
- [ ] 標題層級正確 (h1→h2→h3...)

### 動態效果

- [ ] 動畫效果不過度
- [ ] 支援 prefers-reduced-motion (如可能)

---

## 8. 效能相關視覺問題 (Performance-related Visual Issues)

### 載入

- [ ] 字體 FOUT/FOIT 現象極輕微
- [ ] 未發生版面位移 (CLS)
- [ ] 圖片載入時頁面不會跳動
- [ ] 骨架螢幕 (Skeleton screens) 設定合適 (如適用)

### 動畫

- [ ] 動畫流暢 (60fps)
- [ ] 捲動時無效能問題
- [ ] 轉換效果自然

---

## 優先順序矩陣 (Priority Matrix)

| 優先順序 | 類別 | 範例 |
|----------|----------|----------|
| P0 (緊急) | 功能損壞 | 元件完全重疊、內容消失 |
| P1 (高) | 嚴重 UX 問題 | 文字無法閱讀、按鈕無法操作 |
| P2 (中) | 中度問題 | 對齊問題、間距不一致 |
| P3 (低) | 輕微問題 | 位置輕微差異、顏色微小偏差 |

---

## 驗證工具

### 瀏覽器開發者工具 (DevTools)

- Elements 面板：DOM 與樣式檢查
- Lighthouse：效能與無障礙性稽核
- 裝置工具列：回應式測試

### 無障礙性工具

- axe DevTools
- WAVE
- Color Contrast Analyzer (色彩對比分析器)

### 自動化工具

- Playwright (螢幕截圖比較)
- Percy / Chromatic (視覺迴歸測試)
