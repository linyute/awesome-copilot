## 選擇範本

| 技術型別 | 範本 | 範例 |
|-----------------|----------|----------|
| 用戶端函式庫、NuGet/npm 套件 | SDK/函式庫 | Semantic Kernel, Azure SDK, MSAL |
| Azure 資源 | Azure 服務 | Cosmos DB, Azure Functions, App Service |
| 應用程式開發架構 | 架構/平台 | ASP.NET Core, Blazor, MAUI |
| REST API、協定、規格 | API/協定 | Microsoft Graph, OOXML, FHIR |

## 自訂指引

範本僅是起點。您可以透過以下方式進行自訂：

1. 為該技術的獨特面向**新增章節**
2. **移除不適用**的章節
3. 根據複雜程度**調整深度** (針對複雜技術提供更多概念)
4. 對於不適合放在 SKILL.md 的詳細內容，**新增參考檔案**
5. 對於內聯程式碼片段以外的可執行範例，**新增 sample_codes/**
