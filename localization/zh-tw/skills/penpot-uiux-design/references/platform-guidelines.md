# 平台設計指南參考

## 行動裝置設計基礎

### 螢幕大小

| 裝置              | 大小    | 設計基準     |
| ----------------- | ------- | ------------ |
| iPhone SE         | 375×667 | 小型行動裝置 |
| iPhone 14/15      | 390×844 | 標準行動裝置 |
| iPhone 14 Pro Max | 430×932 | 大型行動裝置 |
| Android 小型      | 360×640 | 最小目標     |
| Android 大型      | 412×915 | 大型 Android |

### 安全區域 (Safe Areas)

```text
┌─────────────────────────────────┐
│ ▓▓▓▓▓▓▓ 狀態列 (Status Bar) ▓▓▓▓ │ 44-47px
├─────────────────────────────────┤
│                                 │
│      安全內容區域 (Safe Area)     │
│                                 │
│                                 │
├─────────────────────────────────┤
│ ▓▓▓▓▓ 主畫面指示條 (Indicator) ▓▓ │ 34px
└─────────────────────────────────┘

```

### 觸控目標

- **最小值：** 44×44pt (iOS) / 48×48dp (Android)
- **建議值：** 所有平台均為 48×48px
- **間距：** 目標之間至少保持 8px

## iOS 人機介面指南 (HIG)

### 設計理念

- **清晰 (Clarity)：** 文字清晰易讀、圖示精確、裝飾細微
- **順應 (Deference)：** UI 協助人們理解內容，絕不與內容競爭
- **深度 (Depth)：** 明確的視覺層次感傳達階層關係

### 導覽模式

| 模式                    | 何時使用                 |
| ----------------------- | ------------------------ |
| 標籤列 (Tab Bar)        | 3-5 個頂層目的地         |
| 導覽列 (Navigation Bar) | 階層式內容               |
| 側邊欄 (Sidebar)        | iPad、內容豐富的應用程式 |
| 搜尋 (Search)           | 內容探索                 |

### 標籤列規格

```text
┌─────────────────────────────────┐
│  🏠    🔍    ➕    💬    👤    │
│  首頁   搜尋  新增   聊天  個人檔案 │ 高度 49pt
└─────────────────────────────────┘

```

- 最多 5 個標籤
- 25×25pt 的圖示，搭配 10pt 的標籤
- 作用中標籤使用填充/著色 (Tint color)
- 非作用中標籤使用灰色

### 導覽列

```text
┌─────────────────────────────────┐
│ ‹ 返回      頁面標題      操作    │ 最小值 44pt
└─────────────────────────────────┘

```

- 左側：返回按鈕或取消
- 中間：標題
- 右側：主要操作（文字或圖示）

| ---------------------- | ---- | ------ |

| 腳註 (Footnote)        | 13pt | 常規   |
| 說明 (Caption)         | 12pt | 常規   |

| 顏色         | 淺色模式      | 深色模式      |
| ------------ | ------------- | ------------- |
| 標籤 (Label) | #000000       | #FFFFFF       |

| 三級標籤     | #3C3C43 @ 30% | #EBEBF5 @ 30% |

| 系統橘色     | #FF9500       | #FF9F0A       |

### iOS 特定模式

- **滑動手勢：** 刪除、封存、操作
- **下拉重新整理：** 標準清單重新整理
- **長按：** 內容選單 (Context menus)
- **觸覺回饋：** 確認操作
- **邊緣滑動：** 返回導覽

## Android Material Design

### Android 設計理念

- **材質即隱喻：** 物理屬性、提升感 (Elevation)
- **大膽、圖形化、有目的性：** 刻意的色彩、排版、空間
- **動態提供意義：** 回饋與連續性

### Android 導覽模式

| 模式                             | 何時使用                 |
| -------------------------------- | ------------------------ |
| 底部導覽 (Bottom Navigation)     | 3-5 個頂層目的地         |
| 導覽側拉選單 (Navigation Drawer) | 5 個以上目的地，較少使用 |
| 導覽欄 (Navigation Rail)         | 平板電腦橫向模式         |
| 索引標籤 (Tabs)                  | 相關內容群組             |

### 底部導覽

```text
┌─────────────────────────────────┐
│  🏠    🔍    📷    💬    👤   │
│  首頁  搜尋   相機   聊天   帳戶   │ 高度 80dp
└─────────────────────────────────┘

```

- 3-5 個目的地
- 24dp 的圖示，搭配 12sp 的標籤
- 作用中：實心圖示 + 主色
- 非作用中：框線圖示 + on-surface 顏色

### 應用程式列 (App Bar)

```text
┌─────────────────────────────────┐
│ ≡  應用程式標題             🔍   │ 高度 64dp
└─────────────────────────────────┘

```

- 左側：導覽圖示（選單或返回）
- 中間：標題（可以左對齊）
- 右側：操作圖示（最多 3 個）

### 懸浮動作按鈕 (FAB)

- **大小：** 標準 56dp，迷你 40dp
- **位置：** 右下角，距離邊緣 16dp
- **用途：** 僅限主要操作
- **行為：** 可隨捲動隱藏

### Material 色彩

| 角色                         | 用途                  |
| ---------------------------- | --------------------- |
| 主要 (Primary)               | 主要品牌顏色          |
| On Primary                   | 主要顏色上的文字/圖示 |
| 主要容器 (Primary Container) | 填充按鈕、啟用狀態    |
| 次要 (Secondary)             | 較不突出的元件        |
| 三級 (Tertiary)              | 對比、平衡            |
| 錯誤 (Error)                 | 錯誤狀態              |
| 表面 (Surface)               | 卡片背景              |
| On Surface                   | 表面上的文字          |
| 框線 (Outline)               | 邊框、分隔線          |

### 提升感（陰影）

| 等級 | 提升值 | 使用案例         |
| ---- | ------ | ---------------- |
| 0    | 0dp    | 平面表面         |
| 1    | 1dp    | 卡片、凸起按鈕   |
| 2    | 3dp    | 具備提升感的卡片 |
| 3    | 6dp    | FAB 靜止狀態     |
| 4    | 8dp    | 對話方塊、選取器 |
| 5    | 12dp   | FAB 按下狀態     |

### Android 特定模式

- **訊息列 (Snackbar)：** 底部簡短回饋
- **底部面板 (Bottom sheet)：** 附加內容/操作
- **標籤 (Chips)：** 篩選、輸入、選擇、操作
- **快速撥號 FAB：** 多個相關操作

## 回應式網頁設計

### 斷點 (Breakpoints)

| 名稱 | 寬度        | 典型裝置     |
| ---- | ----------- | ------------ |
| xs   | <576px      | 行動裝置直向 |
| sm   | 576-767px   | 行動裝置橫向 |
| md   | 768-991px   | 平板電腦     |
| lg   | 992-1199px  | 小型桌面端   |
| xl   | 1200-1399px | 桌面端       |
| xxl  | ≥1400px     | 大型桌面端   |

### 網格系統

- **資料欄：** 標準 12 欄網格
- **欄間距 (Gutters)：** 欄之間 16-24px
- **邊距 (Margins)：** 16px（行動端）到 64px（桌面端）
- **最大內容寬度：** 1200-1440px

### 回應式排版

```text
行動端（基準）：
  內文：16px
  H1：28-32px
  H2：22-24px
  
平板電腦：
  內文：16px
  H1：32-40px
  H2：24-28px
  
桌面端：
  內文：16-18px
  H1：40-56px
  H2：28-36px

```

### 行動優先原則 (Mobile-First Approach)

1. 先針對最小螢幕進行設計
2. 為較大螢幕增加複雜度
3. 內容優先順序：什麼是必要的？
4. 效能：針對行動端進行最小化
5. 觸控優先互動

### 回應式模式

| 模式                  | 描述                       |
| --------------------- | -------------------------- |
| 堆疊 (Stack)          | 行動端將欄變為列           |
| 重排 (Reflow)         | 內容根據優先順序重新排序   |
| 揭露 (Reveal)         | 在較大尺寸顯示更多內容     |
| 側拉選單 (Off-canvas) | 導覽選單在行動端從側面滑入 |
| 縮放 (Scale)          | 元素按比例縮放             |

## 桌面端應用程式

### 視窗邊框 (Window Chrome)

```text
┌─────────────────────────────────────────┐
│ ● ● ●   應用程式標題           ─ □ ×      │ 標題列
├────────┬────────────────────────────────┤
│ 側邊欄  │ 內容區域                        │
│        │                                │
│        │                                │
│        │                                │
│        ├────────────────────────────────┤
│        │ 狀態列                          │
└────────┴────────────────────────────────┘

```

### 鍵盤優先設計

- 所有操作均可透過鍵盤存取
- 可見的鍵盤快速鍵
- 為索引標籤順序進行焦點管理
- 搜尋/指令面板 (Cmd/Ctrl+K)

### 懸停狀態 (Hover States)

桌面端具備懸停功能（行動端則無）：

- 懸停時顯示附加資訊
- 點擊前預覽操作
- 針對僅含圖示按鈕的工具提示 (Tooltips)
- 懸停時顯示下拉選單

### 密集資訊

桌面端允許：

- 較小的觸控目標（最小 32px）
- 更多可見資訊
- 複雜的表格和資料網格
- 多欄佈局
- 並排比較

## 跨平台考量

### 共享原則

- 一致的品牌識別
- 相同的核心使用者流程
- 同步的資料/狀態
- 熟悉的資訊架構

### 平台特定改編

| 面向     | iOS                    | Android        | 網頁 (Web)    |
| -------- | ---------------------- | -------------- | ------------- |
| 返回     | 左側導覽列             | 向左滑動或手勢 | 瀏覽器返回    |
| 主要操作 | 右側導覽列             | FAB            | 右上角按鈕    |
| 清單     | 滑動操作               | 長按           | 懸停操作      |
| 選單     | 操作表 (Action sheets) | 底部面板       | 下拉/內容選單 |
| 警示     | 居中互動視窗           | 居中互動視窗   | 各種位置      |

### 跨平台設計權杖 (Design Tokens)

建立與平台無關的權杖：

```text
// 間距
spacing-sm: 8
spacing-md: 16
spacing-lg: 24

// 這些會映射到平台單位
iOS: 點 (pt)
Android: 密度獨立像素 (dp)
網頁: 像素 (px) 或 rem

```
