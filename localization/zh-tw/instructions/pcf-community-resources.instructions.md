# PCF 社群資源

Power Apps Component Framework 擁有一個充滿活力的社群，致力於建立和分享資源、工具和知識。本指南提供主要社群資源的連結。

## PCF 圖庫

**[PCF 圖庫](https://pcf.gallery)**

PCF 圖庫是發現、分享和學習 PCF 元件的中心樞紐。

**您會發現：**
- 社群建立的元件
- 元件展示和螢幕截圖
- 原始程式碼連結
- 安裝說明
- 元件評分和評論
- 搜尋和篩選功能

**使用方法：**
- 按類別瀏覽元件
- 搜尋特定功能
- 下載元件以用於您的專案
- 提交您自己的元件以與社群分享
- 從真實世界的元件實作中學習

## 社群最佳實務

1. **分享您的工作**：將元件和知識貢獻回社群
2. **提供回饋**：報告問題並提出改進建議
3. **良好文件化**：在您的元件中包含清晰的文件
4. **徹底測試**：在分享之前確保元件在各個平台上都能運作
5. **遵循標準**：使用既定的模式和命名約定
6. **樂於助人**：回答問題並幫助其他開發人員

## 貢獻到 PCF 圖庫

若要將您的元件新增到 PCF 圖庫：

1. 建立一個文件齊全的元件
2. 在目標平台上進行測試
3. 準備螢幕截圖和演示
4. 提交到 pcf.gallery
5. 包含原始程式碼連結 (建議使用 GitHub)
6. 提供清晰的安裝說明
