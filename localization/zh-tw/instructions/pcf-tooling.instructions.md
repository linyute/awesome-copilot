# 取得 Power Apps Component Framework 的工具

使用 Microsoft Power Platform CLI (命令列介面) 來建立、偵錯和部署使用 Power Apps Component Framework 的程式碼元件。Microsoft Power Platform CLI 讓開發人員能夠快速建立程式碼元件。未來，它將擴展以支援額外的開發和應用程式生命週期管理 (ALM) 體驗。

更多資訊：[安裝 Microsoft Power Platform CLI](https://learn.microsoft.com/en-us/power-apps/developer/data-platform/powerapps-cli)

> **重要事項**：若要使用 Microsoft Power Platform CLI 部署您的程式碼元件，您必須擁有具備系統管理員或系統自訂員權限的 Microsoft Dataverse 環境。
