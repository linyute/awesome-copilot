## 集合用法

### C# MCP 伺服器專家

推薦

此聊天模式為使用 C# 建立 MCP 伺服器提供專家指導。

此聊天模式非常適合：
- 建立新的 MCP 伺服器專案
- 實作工具和提示
- 偵錯協定問題
- 優化伺服器效能
- 學習 MCP 最佳實踐

為了獲得最佳結果，請考慮：
- 使用指示檔案為所有 Copilot 互動設定上下文
- 使用提示產生初始專案結構
- 切換到專家聊天模式以獲得詳細的實作協助
- 提供有關您需要哪些工具或功能的具體細節
