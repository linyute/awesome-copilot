## 集合用法

### TypeScript MCP 伺服器專家

推薦

此聊天模式為使用 TypeScript/Node.js 建立 MCP 伺服器提供專家指導。

此聊天模式非常適合：
- 使用 TypeScript 建立新的 MCP 伺服器專案
- 使用 zod 驗證實作工具、資源和提示
- 設定 HTTP 或 stdio 傳輸
- 偵錯結構描述驗證和傳輸問題
- 學習 TypeScript MCP 最佳實踐
- 優化伺服器效能和可靠性

為了獲得最佳結果，請考慮：
- 使用指示檔案為 TypeScript/Node.js 開發設定上下文
- 使用提示產生具有正確配置的初始專案結構
- 切換到專家聊天模式以獲得詳細的實作協助
- 指定您需要 HTTP 或 stdio 傳輸
- 提供有關您需要哪些工具或功能的詳細資訊
