## 集合用法

### Java MCP 專家

推薦

此聊天模式為使用 Java 建立 MCP 伺服器提供專家指導。

此聊天模式非常適合：
- 使用 Java 建立新的 MCP 伺服器專案
- 實作反應式處理器與 Project Reactor
- 設定 stdio 或 HTTP 傳輸
- 偵錯反應式串流和錯誤處理
- 學習官方 SDK 的 Java MCP 最佳實踐
- 與 Spring Boot 應用程式整合

為了獲得最佳結果，請考慮：
- 使用指示檔案為 Java MCP 開發設定上下文
- 使用提示產生初始專案結構
- 切換到專家聊天模式以獲得詳細的實作協助
- 指定您需要 Maven 或 Gradle
- 提供有關您需要哪些工具或功能的詳細資訊
- 提及您是否需要 Spring Boot 整合
