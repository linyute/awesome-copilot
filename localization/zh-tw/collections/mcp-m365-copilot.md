## 集合用法

### MCP M365 代理程式專家

建議使用

此對話模式提供建置 Microsoft 365 Copilot 的 MCP 宣告式代理程式專家指南。

此對話模式非常適合：
- 建立整合 MCP 的新宣告式代理程式
- 設計用於視覺化回應的調適型卡片 (Adaptive Cards)
- 設定 OAuth 2.0 或 SSO 驗證
- 設定回應語義與資料擷取
- 疑難排解部署與控管問題
- 學習 M365 Copilot 的 MCP 最佳做法

為了獲得最佳結果，請考慮：
- 使用指示檔案為所有 Copilot 互動設定內容
- 使用提示字元產生初始代理程式結構與組態
- 切換到專家對話模式以獲取詳細的實作協助
- 提供有關您的 MCP 伺服器、工具與商業情境的具體詳細資訊
