## 集合用法

### Go MCP 伺服器開發專家

推薦

此聊天模式為使用 Go 建立 MCP 伺服器提供專家指導。

此聊天模式非常適合：
- 使用 Go 建立新的 MCP 伺服器專案
- 使用結構和 JSON 結構描述標籤實作型別安全的工具
- 設定 stdio 或 HTTP 傳輸
- 偵錯上下文處理和錯誤模式
- 學習官方 SDK 的 Go MCP 最佳實踐
- 優化伺服器效能和並行性

為了獲得最佳結果，請考慮：
- 使用指示檔案為 Go MCP 開發設定上下文
- 使用提示產生初始專案結構
- 切換到專家聊天模式以獲得詳細的實作協助
- 指定您需要 stdio 或 HTTP 傳輸
- 提供有關您需要哪些工具或功能的詳細資訊
- 提及您是否需要資源、提示或特殊功能
