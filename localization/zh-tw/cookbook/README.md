# GitHub Copilot Cookbook

GitHub Copilot 工具與功能的實用食譜與範例集合。每份食譜都提供專注、可複製貼上的程式碼片段以及真實世界的範例，協助您完成常見任務。

## Cookbook 內容

此 Cookbook 依據工具或產品進行組織，並在適用時按語言收集食譜：

### GitHub Copilot SDK

適用於在多種語言中使用 GitHub Copilot SDK 進行建構的即用型食譜。

- **[Copilot SDK Cookbook](copilot-sdk/)** - .NET、Go、Node.js 與 Python 的食譜
  - 錯誤處理、工作階段管理、檔案操作等
  - 每種語言的可執行範例
  - 最佳實踐與完整的實作指南

## 開始使用

1. 瀏覽與您想要建構的內容相符的工具或產品資料夾
2. 尋找解決您使用案例的食譜
3. 複製程式碼片段或查看 `recipe/` 子資料夾以獲取完整的可執行範例
4. 參考特定語言的文件以獲取設定與執行說明

## 計劃中的擴充

此 Cookbook 旨在隨 GitHub Copilot 生態系統一同成長。未來的補充可能包括以下食譜：

- 其他 Copilot 工具與整合
- 進階模式與工作流
- 與外部服務和 API 的整合
- 特定語言的最佳化與最佳實踐
