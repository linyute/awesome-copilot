# GitHub Copilot Cookbook

GitHub Copilot 相關工具與功能的實用食譜（recipe）與範例集合。每份食譜都提供了專注且可直接複製貼上的程式碼片段以及真實世界的範例，協助您完成常見的工作。

## Cookbook 內容

本 Cookbook 按工具或產品進行組織，並在適用的情況下按程式語言收集食譜：

### GitHub Copilot SDK

可用於跨多種語言建構 GitHub Copilot SDK 的即用型食譜。

- **[Copilot SDK Cookbook](copilot-sdk/)** - 適用於 .NET、Go、Node.js 與 Python 的食譜
  - 錯誤處理、工作階段（session）管理、檔案操作等
  - 每種語言的可執行範例
  - 最佳實踐與完整的實作指南

## 入門指南

1. 瀏覽與您想要建構的內容相匹配的工具或產品資料夾
2. 尋找解決您使用案例的食譜
3. 複製程式碼片段或查看 `recipe/` 子資料夾以獲取完整的可執行範例
4. 參考特定語言的文件以了解設定與執行說明

## 計畫擴展

本 Cookbook 旨在與 GitHub Copilot 生態系統一同成長。未來的補充可能包括以下內容的食譜：

- 其他 Copilot 工具與整合
- 進階模式與工作流程
- 與外部服務與 API 的整合
- 特定語言的優化與最佳實踐
