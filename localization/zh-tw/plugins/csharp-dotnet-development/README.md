# C# .NET 開發延伸模組

適用於 C# 和 .NET 開發的基本提示、指令和對話模式，包含測試、說明文件和最佳做法。

## 安裝

```bash
# 使用 Copilot CLI
copilot plugin install csharp-dotnet-development@awesome-copilot
```

## 包含內容

### 指令 (斜線指令)

| 指令 | 描述 |
|---------|-------------|
| `/csharp-dotnet-development:csharp-async` | 獲取 C# 非同步程式設計的最佳做法 |
| `/csharp-dotnet-development:aspnet-minimal-api-openapi` | 建立具有正確 OpenAPI 說明文件的 ASP.NET Minimal API 端點 |
| `/csharp-dotnet-development:csharp-xunit` | 獲取 XUnit 單元測試的最佳做法，包含資料驅動測試 |
| `/csharp-dotnet-development:dotnet-best-practices` | 確保 .NET/C# 程式碼符合解決方案/專案的最佳做法。 |
| `/csharp-dotnet-development:dotnet-upgrade` | 用於全面 .NET 框架升級分析和執行的即用型提示 |

### Agent

| Agent | 描述 |
|-------|-------------|
| `expert-dotnet-software-engineer` | 使用現代軟體設計模式提供專家級 .NET 軟體工程指導。 |

## 來源

此延伸模組是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，這是一個社群驅動的 GitHub Copilot 擴充功能集合。
