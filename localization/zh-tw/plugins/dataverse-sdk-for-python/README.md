# Dataverse SDK for Python 延伸模組

使用 Microsoft Dataverse 建置生產級 Python 整合的全面集合。包含官方文件、最佳做法、進階功能、檔案操作和程式碼產生提示。

## 安裝

```bash
# 使用 Copilot CLI
copilot plugin install dataverse-sdk-for-python@awesome-copilot
```

## 包含內容

### 指令 (斜線指令)

| 指令 | 描述 |
|---------|-------------|
| `/dataverse-sdk-for-python:dataverse-python-quickstart` | 使用官方模式產生 Python SDK 設定 + CRUD + 批次 + 分頁程式碼片段。 |
| `/dataverse-sdk-for-python:dataverse-python-advanced-patterns` | 使用進階模式、錯誤處理和最佳化技術為 Dataverse SDK 產生生產環境程式碼。 |
| `/dataverse-sdk-for-python:dataverse-python-production-code` | 使用 Dataverse SDK 產生生產級 Python 程式碼，包含錯誤處理、最佳化和最佳做法 |
| `/dataverse-sdk-for-python:dataverse-python-usecase-builder` | 為特定的 Dataverse SDK 使用案例產生完整的解決方案，並提供架構建議 |

## 來源

此延伸模組是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，這是一個社群驅動的 GitHub Copilot 擴充功能集合。
