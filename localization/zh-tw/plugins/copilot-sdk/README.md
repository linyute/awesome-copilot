# Copilot SDK 延伸模組

使用 GitHub Copilot SDK 跨多種程式語言建置應用程式。包含適用於 C#、Go、Node.js/TypeScript 和 Python 的全面指令，協助您建立 AI 驅動的應用程式。

## 安裝

```bash
# 使用 Copilot CLI
copilot plugin install copilot-sdk@awesome-copilot
```

## 包含內容

### 技能

| 技能 | 描述 |
|-------|-------------|
| `SKILL.md` | 使用 GitHub Copilot SDK 建置 Agentic 應用程式。用於在應用程式中嵌入 AI Agent、建立自訂工具、實作串流回應、管理對話、連接到 MCP 伺服器或建立自訂 Agent。觸發於 Copilot SDK、GitHub SDK、Agentic 應用程式、嵌入 Copilot、可程式化 Agent、MCP 伺服器、自訂 Agent。 |

## 來源

此延伸模組是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，這是一個社群驅動的 GitHub Copilot 擴充功能集合。
