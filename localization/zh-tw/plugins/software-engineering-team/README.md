# 軟體工程團隊延伸模組

7 位專業 Agent，涵蓋從 UX 設計、架構到安全性與 DevOps 的完整軟體開發生命週期。

## 安裝

```bash
# 使用 Copilot CLI
copilot plugin install software-engineering-team@awesome-copilot
```

## 包含內容

### Agent

| Agent | 描述 |
|-------|-------------|
| `se-ux-ui-designer` | 針對 Figma 和設計工作流進行 Jobs-to-be-Done 分析、使用者旅程對應和 UX 研究產物 |
| `se-technical-writer` | 用於建立開發者文件、技術部落格、教學課程和教育內容的技術寫作專家 |
| `se-gitops-ci-specialist` | 專注於使部署變得平淡且可靠的 DevOps 專家，負責 CI/CD 管線、部署偵錯和 GitOps 工作流 |
| `se-product-manager-advisor` | 用於建立 GitHub Issue、將商業價值與使用者需求對齊，以及制定資料驅動產品決策的產品管理指引 |
| `se-responsible-ai-code` | 負責任 AI 專家，透過預防偏見、輔助功能合規性、倫理性開發和包容性設計，確保 AI 為每個人服務 |
| `se-system-architecture-reviewer` | 具有 Well-Architected 框架的系統架構檢閱專家，針對 AI 和分散式系統進行設計驗證與延展性分析 |
| `se-security-reviewer` | 具有 OWASP Top 10、零信任、LLM 安全性和企業安全性標準的安全性導向程式碼檢閱專家 |

## 來源

此延伸模組是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，這是一個社群驅動的 GitHub Copilot 擴充功能集合。
