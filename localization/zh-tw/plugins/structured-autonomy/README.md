# 結構化自主 (Structured Autonomy) 延伸模組

卓越的規劃，精簡的實作

## 安裝

```bash
# 使用 Copilot CLI
copilot plugin install github/awesome-copilot/plugins/structured-autonomy
```

## 包含內容

### 指令 (斜線指令)

| 指令 | 描述 |
|---------|-------------|
| `/structured-autonomy:structured-autonomy-plan` | 結構化自主規劃 |
| `/structured-autonomy:structured-autonomy-generate` | 結構化自主產生 |
| `/structured-autonomy:structured-autonomy-implement` | 結構化自主實作 |

## 來源

此延伸模組是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，這是一個社群驅動的 GitHub Copilot 擴充功能集合。
