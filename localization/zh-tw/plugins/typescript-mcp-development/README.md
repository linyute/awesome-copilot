# TypeScript MCP 伺服器開發延伸模組

使用官方 SDK 在 TypeScript/Node.js 中建置 Model Context Protocol (MCP) 伺服器的完整工具包。包含最佳做法指令、用於產生伺服器的提示以及用於指引的專家對話模式。

## 安裝

```bash
# 使用 Copilot CLI
copilot plugin install typescript-mcp-development@awesome-copilot
```

## 包含內容

### 指令 (斜線指令)

| 指令 | 描述 |
|---------|-------------|
| `/typescript-mcp-development:typescript-mcp-server-generator` | 使用工具、資源和正確的設定在 TypeScript 中產生完整的 MCP 伺服器專案 |

### Agent

| Agent | 描述 |
|-------|-------------|
| `typescript-mcp-expert` | 專門用於開發 TypeScript Model Context Protocol (MCP) 伺服器的專家助手 |

## 來源

此延伸模組是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，這是一個社群驅動的 GitHub Copilot 擴充功能集合。
