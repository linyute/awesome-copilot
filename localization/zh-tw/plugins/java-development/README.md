# Java 開發延伸模組

適用於 Java 開發（包括 Spring Boot、Quarkus、測試、說明文件和最佳做法）的提示和指令全面集合。

## 安裝

```bash
# 使用 Copilot CLI
copilot plugin install java-development@awesome-copilot
```

## 包含內容

### 指令 (斜線指令)

| 指令 | 描述 |
|---------|-------------|
| `/java-development:java-docs` | 確保 Java 類型已編寫 Javadoc 註解，並遵循說明文件的最佳做法。 |
| `/java-development:java-junit` | 獲取 JUnit 5 單元測試的最佳做法，包含資料驅動測試 |
| `/java-development:java-springboot` | 獲取使用 Spring Boot 開發應用程式的最佳做法。 |
| `/java-development:create-spring-boot-java-project` | 建立 Spring Boot Java 專案骨架 |

## 來源

此延伸模組是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，這是一個社群驅動的 GitHub Copilot 擴充功能集合。
