# Java MCP 伺服器開發延伸模組

使用具有反應式串流 (Reactive Streams) 和 Spring Boot 整合的官方 MCP Java SDK，建置 Java Model Context Protocol 伺服器的完整工具包。

## 安裝

```bash
# 使用 Copilot CLI
copilot plugin install java-mcp-development@awesome-copilot
```

## 包含內容

### 指令 (斜線指令)

| 指令 | 描述 |
|---------|-------------|
| `/java-mcp-development:java-mcp-server-generator` | 使用具有反應式串流和選用 Spring Boot 整合的官方 MCP Java SDK，在 Java 中產生完整的 Model Context Protocol 伺服器專案。 |

### Agent

| Agent | 描述 |
|-------|-------------|
| `java-mcp-expert` | 使用反應式串流、官方 MCP Java SDK 和 Spring Boot 整合，建置 Java Model Context Protocol 伺服器的專家助手。 |

## 來源

此延伸模組是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，這是一個社群驅動的 GitHub Copilot 擴充功能集合。
