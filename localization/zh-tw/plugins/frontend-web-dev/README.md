# 前端網頁開發延伸模組

適用於現代前端網頁開發（包括 React、Angular、Vue、TypeScript 和 CSS 框架）的基本提示、指令和對話模式。

## 安裝

```bash
# 使用 Copilot CLI
copilot plugin install frontend-web-dev@awesome-copilot
```

## 包含內容

### 指令 (斜線指令)

| 指令 | 描述 |
|---------|-------------|
| `/frontend-web-dev:playwright-explore-website` | 使用 Playwright MCP 進行測試用的網站探索 |
| `/frontend-web-dev:playwright-generate-test` | 使用 Playwright MCP 根據場景產生 Playwright 測試 |

### Agent

| Agent | 描述 |
|-------|-------------|
| `expert-react-frontend-engineer` | 專家級 React 19.2 前端工程師，專精於現代 Hook、Server Component、Action、TypeScript 和效能最佳化 |
| `electron-angular-native` | 為具有 Node.js 後端 (main)、Angular 前端 (render) 和原生整合層 (例如 AppleScript、Shell 或原生工具) 的 Electron 應用程式量身打造的程式碼檢閱模式。此處不檢閱其他存放庫中的服務。 |

## 來源

此延伸模組是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，這是一個社群驅動的 GitHub Copilot 擴充功能集合。
