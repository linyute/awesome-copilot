# OpenAPI 轉應用程式 - C# .NET 延伸模組

根據 OpenAPI 規格產生生產級 .NET 應用程式。包含 ASP.NET Core 專案架構、控制器產生、Entity Framework 整合以及 C# 最佳做法。

## 安裝

```bash
# 使用 Copilot CLI
copilot plugin install openapi-to-application-csharp-dotnet@awesome-copilot
```

## 包含內容

### 指令 (斜線指令)

| 指令 | 描述 |
|---------|-------------|
| `/openapi-to-application-csharp-dotnet:openapi-to-application-code` | 根據 OpenAPI 規格產生完整的生產級應用程式 |

### Agent

| Agent | 描述 |
|-------|-------------|
| `openapi-to-application` | 根據 OpenAPI 規格產生可用應用程式的專家助手 |

## 來源

此延伸模組是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，這是一個社群驅動的 GitHub Copilot 擴充功能集合。
