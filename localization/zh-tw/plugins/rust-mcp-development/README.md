# Rust MCP 伺服器開發延伸模組

使用具有 async/await、程序巨集 (Procedural Macro) 和型別安全實作的官方 rmcp SDK，在 Rust 中建置高效能的 Model Context Protocol 伺服器。

## 安裝

```bash
# 使用 Copilot CLI
copilot plugin install rust-mcp-development@awesome-copilot
```

## 包含內容

### 指令 (斜線指令)

| 指令 | 描述 |
|---------|-------------|
| `/rust-mcp-development:rust-mcp-server-generator` | 使用官方 rmcp SDK 產生具有工具、提示、資源和測試的完整 Rust Model Context Protocol 伺服器專案 |

### Agent

| Agent | 描述 |
|-------|-------------|
| `rust-mcp-expert` | 使用具有 Tokio 非同步執行階段的 rmcp SDK 進行 Rust MCP 伺服器開發的專家助手 |

## 來源

此延伸模組是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，這是一個社群驅動的 GitHub Copilot 擴充功能集合。
