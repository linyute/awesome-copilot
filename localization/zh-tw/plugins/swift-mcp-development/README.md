# Swift MCP 伺服器開發延伸模組

使用具有現代並行功能 (Concurrency) 的官方 MCP Swift SDK，在 Swift 中建置 Model Context Protocol 伺服器的全面集合。

## 安裝

```bash
# 使用 Copilot CLI
copilot plugin install swift-mcp-development@awesome-copilot
```

## 包含內容

### 指令 (斜線指令)

| 指令 | 描述 |
|---------|-------------|
| `/swift-mcp-development:swift-mcp-server-generator` | 使用官方 MCP Swift SDK 套件在 Swift 中產生完整的 Model Context Protocol 伺服器專案。 |

### Agent

| Agent | 描述 |
|-------|-------------|
| `swift-mcp-expert` | 使用現代並行功能和官方 MCP Swift SDK 建置 Model Context Protocol 伺服器的專家助手。 |

## 來源

此延伸模組是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，這是一個社群驅動的 GitHub Copilot 擴充功能集合。
