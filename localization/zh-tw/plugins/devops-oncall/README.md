# DevOps On-Call 延伸模組

一組集中的提示、指令和對話模式，協助分類突發事件，並使用 DevOps 工具和 Azure 資源快速回應。

## 安裝

```bash
# 使用 Copilot CLI
copilot plugin install devops-oncall@awesome-copilot
```

## 包含內容

### 指令 (斜線指令)

| 指令 | 描述 |
|---------|-------------|
| `/devops-oncall:azure-resource-health-diagnose` | 分析 Azure 資源健康狀況，從記錄和遙測中診斷問題，並為識別出的問題建立修復計劃。 |
| `/devops-oncall:multi-stage-dockerfile` | 為任何語言或框架建立最佳化的多階段 Dockerfile |

### Agent

| Agent | 描述 |
|-------|-------------|
| `azure-principal-architect` | 使用 Azure Well-Architected Framework 原則和 Microsoft 最佳做法提供專家級 Azure 首席架構師指導。 |

## 來源

此延伸模組是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，這是一個社群驅動的 GitHub Copilot 擴充功能集合。
