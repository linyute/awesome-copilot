# Awesome Copilot 延伸模組

Meta Prompt 可協助您探索和產生精選的 GitHub Copilot Agent、集合、指令、提示和技能。

## 安裝

```bash
# 使用 Copilot CLI
copilot plugin install awesome-copilot@awesome-copilot
```

## 包含內容

### 指令 (斜線指令)

| 指令 | 描述 |
|---------|-------------|
| `/awesome-copilot:suggest-awesome-github-copilot-collections` | 根據目前存放庫內容和對話記錄，從 awesome-copilot 存放庫建議相關的 GitHub Copilot 集合，提供集合資產的自動下載和安裝，並識別需要更新的過時集合資產。 |
| `/awesome-copilot:suggest-awesome-github-copilot-instructions` | 根據目前存放庫內容和對話記錄，從 awesome-copilot 存放庫建議相關的 GitHub Copilot 指令檔案，避免與此存放庫中的現有指令重複，並識別需要更新的過時指令。 |
| `/awesome-copilot:suggest-awesome-github-copilot-prompts` | 根據目前存放庫內容和對話記錄，從 awesome-copilot 存放庫建議相關的 GitHub Copilot 提示檔案，避免與此存放庫中的現有提示重複，並識別需要更新的過時提示。 |
| `/awesome-copilot:suggest-awesome-github-copilot-agents` | 根據目前存放庫內容和對話記錄，從 awesome-copilot 存放庫建議相關的 GitHub Copilot 自訂 Agent 檔案，避免與此存放庫中的現有自訂 Agent 重複，並識別需要更新的過時 Agent。 |

### Agent

| Agent | 描述 |
|-------|-------------|
| `meta-agentic-project-scaffold` | Meta Agent 專案建立助手，協助使用者有效地建立和管理專案工作流。 |

## 來源

此延伸模組是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，這是一個社群驅動的 GitHub Copilot 擴充功能集合。
