# CAST Imaging Agent 延伸模組

專門用於軟體分析、影響評估、結構品質建議以及使用 CAST Imaging 進行架構檢閱的全面 Agent 集合。

## 安裝

```bash
# 使用 Copilot CLI
copilot plugin install cast-imaging@awesome-copilot
```

## 包含內容

### Agent

| Agent | 描述 |
|-------|-------------|
| `cast-imaging-software-discovery` | 專門用於透過使用 CAST Imaging 進行靜態程式碼分析，來進行全面軟體應用程式探索和架構對應的 Agent |
| `cast-imaging-impact-analysis` | 專門用於使用 CAST Imaging 進行軟體系統全面變更影響評估和風險分析的 Agent |
| `cast-imaging-structural-quality-advisor` | 專門用於使用 CAST Imaging 識別、分析並提供程式碼品質問題修復指導的 Agent |

## 來源

此延伸模組是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，這是一個社群驅動的 GitHub Copilot 擴充功能集合。
