# Power Platform MCP 連接器開發延伸模組

用於開發具有 Model Context Protocol 整合功能（適用於 Microsoft Copilot Studio）的 Power Platform 自訂連接器的完整工具包

## 安裝

```bash
# 使用 Copilot CLI
copilot plugin install power-platform-mcp-connector-development@awesome-copilot
```

## 包含內容

### 指令 (斜線指令)

| 指令 | 描述 |
|---------|-------------|
| `/power-platform-mcp-connector-development:power-platform-mcp-connector-suite` | 產生具有 MCP 整合功能（適用於 Copilot Studio）的完整 Power Platform 自訂連接器 - 包含結構描述產生、疑難排解和驗證 |
| `/power-platform-mcp-connector-development:mcp-copilot-studio-server-generator` | 產生針對 Copilot Studio 整合最佳化的完整 MCP 伺服器實作，具有正確的結構描述限制和串流 HTTP 支援 |

### Agent

| Agent | 描述 |
|-------|-------------|
| `power-platform-mcp-integration-expert` | 具有 MCP 整合功能（適用於 Copilot Studio）的 Power Platform 自訂連接器開發專家 - 擁有結構描述、協定和整合模式的全面知識 |

## 來源

此延伸模組是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，這是一個社群驅動的 GitHub Copilot 擴充功能集合。
