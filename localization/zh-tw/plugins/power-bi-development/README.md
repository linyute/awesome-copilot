# Power BI 開發延伸模組

全面的 Power BI 開發資源，包含資料模型化、DAX 最佳化、效能調校、視覺化設計、安全性最佳做法，以及用於建置企業級 Power BI 解決方案的 DevOps/ALM 指引。

## 安裝

```bash
# 使用 Copilot CLI
copilot plugin install power-bi-development@awesome-copilot
```

## 包含內容

### 指令 (斜線指令)

| 指令 | 描述 |
|---------|-------------|
| `/power-bi-development:power-bi-dax-optimization` | 全面的 Power BI DAX 公式最佳化提示，用於改善 DAX 計算的效能、可讀性和可維護性。 |
| `/power-bi-development:power-bi-model-design-review` | 全面的 Power BI 資料模型設計檢閱提示，用於評估模型架構、關聯性和最佳化機會。 |
| `/power-bi-development:power-bi-performance-troubleshooting` | 系統化的 Power BI 效能疑難排解提示，用於識別、診斷和解決 Power BI 模型、報表和查詢中的效能問題。 |
| `/power-bi-development:power-bi-report-design-consultation` | Power BI 報表視覺化設計提示，用於建立有效、使用者友善且具備輔助功能的報表，並提供最佳的圖表選擇和版面設計。 |

### Agent

| Agent | 描述 |
|-------|-------------|
| `power-bi-data-modeling-expert` | 使用星狀結構描述原則、關聯性設計和 Microsoft 最佳做法提供專家級 Power BI 資料模型化指導，以實現最佳的模型效能和可用性。 |
| `power-bi-dax-expert` | 使用 Microsoft 最佳做法提供專家級 Power BI DAX 指導，以改善 DAX 公式和計算的效能、可讀性和可維護性。 |
| `power-bi-performance-expert` | 用於疑難排解、監控並改善 Power BI 模型、報表和查詢效能的專家級 Power BI 效能最佳化指導。 |
| `power-bi-visualization-expert` | 使用 Microsoft 最佳做法提供專家級 Power BI 報表設計與視覺化指導，以建立有效、高效能且使用者友善的報表和儀表板。 |
