# 安全性與程式碼品質延伸模組

用於建置安全、可維護且高效能應用程式的安全性框架、輔助功能指引、效能最佳化和程式碼品質最佳做法。

## 安裝

```bash
# 使用 Copilot CLI
copilot plugin install security-best-practices@awesome-copilot
```

## 包含內容

### 指令 (斜線指令)

| 指令 | 描述 |
|---------|-------------|
| `/security-best-practices:ai-prompt-engineering-safety-review` | 全面的 AI 提示工程 (Prompt Engineering) 安全性檢閱與改進提示。分析提示的安全性、偏見、安全性弱點和有效性，同時透過廣泛的框架、測試方法論和教育內容提供詳細的改進建議。 |

## 來源

此延伸模組是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，這是一個社群驅動的 GitHub Copilot 擴充功能集合。
