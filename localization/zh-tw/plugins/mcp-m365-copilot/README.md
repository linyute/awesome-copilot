# 基於 MCP 的 M365 Agent 延伸模組

為 Microsoft 365 Copilot 建置具有 Model Context Protocol 整合的宣告式 Agent 的全面集合

## 安裝

```bash
# 使用 Copilot CLI
copilot plugin install mcp-m365-copilot@awesome-copilot
```

## 包含內容

### 指令 (斜線指令)

| 指令 | 描述 |
|---------|-------------|
| `/mcp-m365-copilot:mcp-create-declarative-agent` | 建立 MCP 宣告式 Agent |
| `/mcp-m365-copilot:mcp-create-adaptive-cards` | 建立 MCP 調適型卡片 (Adaptive Cards) |
| `/mcp-m365-copilot:mcp-deploy-manage-agents` | 部署與管理 MCP Agent |

### Agent

| Agent | 描述 |
|-------|-------------|
| `mcp-m365-agent-expert` | 為 Microsoft 365 Copilot 建置具有 Model Context Protocol 整合的基於 MCP 宣告式 Agent 的專家助手 |

## 來源

此延伸模組是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，這是一個社群驅動的 GitHub Copilot 擴充功能集合。
