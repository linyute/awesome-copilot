# Power Apps Code Apps 開發延伸模組

適用於 Power Apps Code Apps 開發的完整工具包，包含專案架構、開發標準，以及建置具有 Power Platform 整合功能的程式碼優先應用程式的專家指導。

## 安裝

```bash
# 使用 Copilot CLI
copilot plugin install power-apps-code-apps@awesome-copilot
```

## 包含內容

### 指令 (斜線指令)

| 指令 | 描述 |
|---------|-------------|
| `/power-apps-code-apps:power-apps-code-app-scaffold` | 透過 PAC CLI 設定、SDK 整合和連接器配置，搭建完整的 Power Apps Code App 專案架構 |

### Agent

| Agent | 描述 |
|-------|-------------|
| `power-platform-expert` | 提供關於 Code Apps、畫布應用程式、Dataverse、連接器和 Power Platform 最佳做法指導的 Power Platform 專家 |

## 來源

此延伸模組是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，這是一個社群驅動的 GitHub Copilot 擴充功能集合。
