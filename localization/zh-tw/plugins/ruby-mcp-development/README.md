# Ruby MCP 伺服器開發延伸模組

使用具有 Rails 整合支援的官方 MCP Ruby SDK Gem，建置 Ruby Model Context Protocol 伺服器的完整工具包。

## 安裝

```bash
# 使用 Copilot CLI
copilot plugin install ruby-mcp-development@awesome-copilot
```

## 包含內容

### 指令 (斜線指令)

| 指令 | 描述 |
|---------|-------------|
| `/ruby-mcp-development:ruby-mcp-server-generator` | 使用官方 MCP Ruby SDK Gem 在 Ruby 中產生完整的 Model Context Protocol 伺服器專案。 |

### Agent

| Agent | 描述 |
|-------|-------------|
| `ruby-mcp-expert` | 使用具有 Rails 整合功能的官方 MCP Ruby SDK Gem，建置 Ruby Model Context Protocol 伺服器的專家助手。 |

## 來源

此延伸模組是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，這是一個社群驅動的 GitHub Copilot 擴充功能集合。
