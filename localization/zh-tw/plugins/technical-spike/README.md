# 技術研究 (Technical Spike) 延伸模組

用於建立、管理和研究技術研究文件的工具，旨在於進行解決方案的規格制定和實作之前，減少未知因素和假設。

## 安裝

```bash
# 使用 Copilot CLI
copilot plugin install technical-spike@awesome-copilot
```

## 包含內容

### 指令 (斜線指令)

| 指令 | 描述 |
|---------|-------------|
| `/technical-spike:create-technical-spike` | 建立限時的技術研究 (Technical Spike) 文件，用於在實作前研究並解決關鍵的開發決策。 |

### Agent

| Agent | 描述 |
|-------|-------------|
| `research-technical-spike` | 透過詳盡的調查和受控的實驗，系統地研究並驗證技術研究文件。 |

## 來源

此延伸模組是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，這是一個社群驅動的 GitHub Copilot 擴充功能集合。
