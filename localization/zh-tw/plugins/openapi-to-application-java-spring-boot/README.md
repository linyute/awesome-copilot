# OpenAPI 轉應用程式 - Java Spring Boot 延伸模組

根據 OpenAPI 規格產生生產級 Spring Boot 應用程式。包含專案架構、REST 控制器產生、服務層組織以及 Spring Boot 最佳做法。

## 安裝

```bash
# 使用 Copilot CLI
copilot plugin install openapi-to-application-java-spring-boot@awesome-copilot
```

## 包含內容

### 指令 (斜線指令)

| 指令 | 描述 |
|---------|-------------|
| `/openapi-to-application-java-spring-boot:openapi-to-application-code` | 根據 OpenAPI 規格產生完整的生產級應用程式 |

### Agent

| Agent | 描述 |
|-------|-------------|
| `openapi-to-application` | 根據 OpenAPI 規格產生可用應用程式的專家助手 |

## 來源

此延伸模組是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，這是一個社群驅動的 GitHub Copilot 擴充功能集合。
