# PHP MCP 伺服器開發延伸模組

使用具有基於屬性的探索 (Attribute-based Discovery) 的官方 PHP SDK 建置 Model Context Protocol 伺服器的全面資源，包含最佳做法、專案產生和專家協助

## 安裝

```bash
# 使用 Copilot CLI
copilot plugin install php-mcp-development@awesome-copilot
```

## 包含內容

### 指令 (斜線指令)

| 指令 | 描述 |
|---------|-------------|
| `/php-mcp-development:php-mcp-server-generator` | 使用官方 PHP SDK 產生具有工具、資源、提示和測試的完整 PHP Model Context Protocol 伺服器專案 |

### Agent

| Agent | 描述 |
|-------|-------------|
| `php-mcp-expert` | 使用具有基於屬性探索的官方 PHP SDK 進行 PHP MCP 伺服器開發的專家助手 |

## 來源

此延伸模組是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，這是一個社群驅動的 GitHub Copilot 擴充功能集合。
