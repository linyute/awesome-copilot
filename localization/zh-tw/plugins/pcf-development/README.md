# Power Apps 元件框架 (PCF) 開發延伸模組

用於開發模型驅動應用程式和畫布應用程式的 Power Apps 元件框架自訂程式碼元件的完整工具包

## 安裝

```bash
# 使用 Copilot CLI
copilot plugin install pcf-development@awesome-copilot
```
