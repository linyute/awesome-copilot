# PHP MCP 伺服器開發外掛程式 (PHP MCP Server Development Plugin)

使用具備屬性型偵測功能的官方 PHP SDK 建立 Model Context Protocol 伺服器的全面資源，包含最佳實務、專案產生與專家協助

## 安裝 (Installation)

```bash
# 使用 Copilot CLI
copilot plugin install php-mcp-development@awesome-copilot
```

## 包含內容 (What's Included)

### 指令 (斜線指令) (Commands (Slash Commands))

| 指令 | 描述 |
|---------|-------------|
| `/php-mcp-development:php-mcp-server-generator` | 使用官方 PHP SDK 建立包含工具、資源、提示與測試的完整 PHP Model Context Protocol 伺服器專案 |

### 代理程式 (Agents)

| 代理程式 | 描述 |
|-------|-------------|
| `php-mcp-expert` | 使用具備屬性型偵測功能的官方 PHP SDK 進行 PHP MCP 伺服器開發的專家助手 |

## 來源 (Source)

此外掛程式是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，這是一個社群驅動的 GitHub Copilot 擴充功能集合。
