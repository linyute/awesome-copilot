# Java 開發外掛程式 (Java Development Plugin)

Java 開發必備的提示與指引全面集合，包含 Spring Boot、Quarkus、測試、文件與最佳實務。

## 安裝 (Installation)

```bash
# 使用 Copilot CLI
copilot plugin install java-development@awesome-copilot
```

## 包含內容 (What's Included)

### 指令 (斜線指令) (Commands (Slash Commands))

| 指令 | 描述 |
|---------|-------------|
| `/java-development:java-docs` | 確保 Java 類型使用 Javadoc 註解進行記錄，並遵循文件編寫的最佳實務。 |
| `/java-development:java-junit` | 獲取 JUnit 5 單元測試的最佳實務，包含資料驅動測試 |
| `/java-development:java-springboot` | 獲取使用 Spring Boot 開發應用程式的最佳實務。 |
| `/java-development:create-spring-boot-java-project` | 建立 Spring Boot Java 專案骨架 |

## 來源 (Source)

此外掛程式是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，這是一個社群驅動的 GitHub Copilot 擴充功能集合。
