# Phoenix 外掛程式

Phoenix AI 可觀測性技能，用於 LLM 應用程式偵錯、評估與追蹤。包含 CLI 偵錯工具、LLM 評估工作流程以及 OpenInference 追蹤檢測。

## 安裝

```bash
# 使用 Copilot CLI
copilot plugin install phoenix@awesome-copilot
```

## 包含內容

### 技能

| 技能 | 說明 |
|-------|-------------|
| `phoenix-cli` | 使用 Phoenix CLI 對 LLM 應用程式進行偵錯。獲取追蹤、分析錯誤、審查實驗、檢查資料集並查詢 GraphQL API。 |
| `phoenix-evals` | 使用 Phoenix 為 AI/LLM 應用程式建立並執行評估器。 |
| `phoenix-tracing` | 用於 Phoenix AI 可觀測性的 OpenInference 語意慣例與檢測。 |
