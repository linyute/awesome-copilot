# Power Apps 元件架構 (PCF) 開發外掛程式 (Power Apps Component Framework (PCF) Development Plugin)

為模型驅動與畫布應用程式使用 Power Apps 元件架構 (Power Apps Component Framework) 開發自訂程式碼元件的完整工具包

## 安裝 (Installation)

```bash
# 使用 Copilot CLI
copilot plugin install pcf-development@awesome-copilot
```
