# OpenAPI 轉應用程式 - C# .NET 外掛程式 (OpenAPI to Application - C# .NET Plugin)

從 OpenAPI 規格產生生產等級的 .NET 應用程式。包含 ASP.NET Core 專案骨架 (scaffolding)、控制器產生、Entity Framework 整合以及 C# 最佳實務。

## 安裝 (Installation)

```bash
# 使用 Copilot CLI
copilot plugin install openapi-to-application-csharp-dotnet@awesome-copilot
```

## 包含內容 (What's Included)

### 指令 (斜線指令) (Commands (Slash Commands))

| 指令 | 描述 |
|---------|-------------|
| `/openapi-to-application-csharp-dotnet:openapi-to-application-code` | 從 OpenAPI 規格產生一個完整的生產等級應用程式 |

### 代理程式 (Agents)

| 代理程式 | 描述 |
|-------|-------------|
| `openapi-to-application` | 從 OpenAPI 規格產生可用應用程式的專家助手 |

## 來源 (Source)

此外掛程式是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，這是一個社群驅動的 GitHub Copilot 擴充功能集合。
