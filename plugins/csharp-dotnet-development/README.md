# C# .NET 開發外掛程式 (C# .NET Development Plugin)

包含測試、文件與最佳實務等 C# 與 .NET 開發必備的提示、指引與聊天模式。

## 安裝 (Installation)

```bash
# 使用 Copilot CLI
copilot plugin install csharp-dotnet-development@awesome-copilot
```

## 包含內容 (What's Included)

### 指令 (斜線指令) (Commands (Slash Commands))

| 指令 | 描述 |
|---------|-------------|
| `/csharp-dotnet-development:csharp-async` | 獲取 C# 非同步程式設計的最佳實務 |
| `/csharp-dotnet-development:aspnet-minimal-api-openapi` | 建立具備正確 OpenAPI 文件的 ASP.NET Minimal API 端點 |
| `/csharp-dotnet-development:csharp-xunit` | 獲取 XUnit 單元測試的最佳實務，包含資料驅動測試 |
| `/csharp-dotnet-development:csharp-nunit` | 獲取 NUnit 單元測試的最佳實務，包含資料驅動測試 |
| `/csharp-dotnet-development:csharp-mstest` | 獲取 MSTest 3.x/4.x 單元測試的最佳實務，包含現代化斷言 API 與資料驅動測試 |
| `/csharp-dotnet-development:csharp-tunit` | 獲取 TUnit 單元測試的最佳實務，包含資料驅動測試 |
| `/csharp-dotnet-development:dotnet-best-practices` | 確保 .NET/C# 程式碼符合解決方案/專案的最佳實務。 |
| `/csharp-dotnet-development:dotnet-upgrade` | 用於全面 .NET 架構升級分析與執行的即用型提示 |

### 代理程式 (Agents)

| 代理程式 | 描述 |
|-------|-------------|
| `expert-dotnet-software-engineer` | 使用現代化軟體設計模式提供專家級 .NET 軟體工程指引。 |
