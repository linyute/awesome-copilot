# FlowStudio Power Automate 擴充功能

透過 FlowStudio MCP 伺服器管理 Power Automate 雲端流程的完整工具組。使用 AI 代理程式連線、偵錯並建構/部署流程。

需要 FlowStudio MCP 訂閱 — 請參閱 https://flowstudio.app

## 安裝 (Installation)

```bash
# 使用 Copilot CLI
copilot plugin install flowstudio-power-automate@awesome-copilot
```

## 包含內容 (What's Included)

### 技能 (Skills)

| 技能 | 說明 |
|-------|-------------|
| `flowstudio-power-automate-mcp` | 透過 FlowStudio MCP 伺服器進行 Power Automate 雲端流程的核心連線設定、工具探索及 CRUD 作業。 |
| `flowstudio-power-automate-debug` | 調查並修正失敗的 Power Automate 雲端流程執行記錄的逐步診斷工作流程。 |
| `flowstudio-power-automate-build` | 根據自然語言描述建構、架構並部署 Power Automate 雲端流程，並隨附動作模式範本。 |

## 開始使用 (Getting Started)

1. 安裝擴充功能
2. 於 https://flowstudio.app 訂閱 FlowStudio MCP
3. 使用來自您工作區的 JWT 設定您的 MCP 連線
4. 要求 Copilot 列出您的流程、偵錯失敗原因或建構新流程
