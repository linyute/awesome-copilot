# Power BI 開發外掛程式 (Power BI Development Plugin)

全面的 Power BI 開發資源，包含資料模型建構、DAX 最佳化、效能調整、視覺化設計、安全性最佳實務，以及建立企業級 Power BI 解決方案的 DevOps/ALM 指引。

## 安裝 (Installation)

```bash
# 使用 Copilot CLI
copilot plugin install power-bi-development@awesome-copilot
```

## 包含內容 (What's Included)

### 指令 (斜線指令) (Commands (Slash Commands))

| 指令 | 描述 |
|---------|-------------|
| `/power-bi-development:power-bi-dax-optimization` | 全面的 Power BI DAX 公式最佳化提示，用於改善 DAX 計算的效能、可讀性與可維護性。 |
| `/power-bi-development:power-bi-model-design-review` | 全面的 Power BI 資料模型設計檢閱提示，用於評估模型架構、關聯性與最佳化機會。 |
| `/power-bi-development:power-bi-performance-troubleshooting` | 系統化的 Power BI 效能疑難排解提示，用於識別、診斷與解決 Power BI 模型、報表與查詢中的效能問題。 |
| `/power-bi-development:power-bi-report-design-consultation` | Power BI 報表視覺化設計提示，用於透過最佳圖表選擇與版面設計，建立有效、使用者友善且具備協助工具的報表。 |

### 代理程式 (Agents)

| 代理程式 | 描述 |
|-------|-------------|
| `power-bi-data-modeling-expert` | 使用星狀結構 (star schema) 原則、關聯性設計與 Microsoft 最佳實務提供專家級 Power BI 資料模型建構指引，以達成最佳模型效能與可用性。 |
| `power-bi-dax-expert` | 使用 Microsoft 最佳實務提供專家級 Power BI DAX 指引，以提升 DAX 公式與計算的效能、可讀性與可維護性。 |
| `power-bi-performance-expert` | 專家級 Power BI 效能最佳化指引，用於對 Power BI 模型、報表與查詢進行疑難排解、監控與效能改善。 |
| `power-bi-visualization-expert` | 專家級 Power BI 報表設計與視覺化指引，使用 Microsoft 最佳實務建立有效、高效能且使用者友善的報表與儀表板。 |
