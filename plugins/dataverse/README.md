# Dataverse MCP

用於 Microsoft Dataverse 整合的全面集合。包含引導您為 GitHub Copilot 設定 Dataverse MCP 伺服器的 MCP 設定指令。

## 安裝 (Installation)

```bash
# 使用 Copilot CLI
copilot plugin install dataverse@awesome-copilot
```

## 包含內容 (What's Included)

### 技能 (Skills)

| 技能 | 描述 |
|-------|-------------|
| `/dataverse:mcp-configure` | 使用全域或專案範圍設定為 GitHub Copilot 設定 Dataverse MCP 伺服器。 |

## 來源 (Source)

此外掛程式是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，這是一個社群驅動的 GitHub Copilot 擴充功能集合。
