# Swift MCP 伺服器開發外掛程式 (Swift MCP Server Development Plugin)

使用官方 MCP Swift SDK 並搭配現代化並行功能，在 Swift 中建立 Model Context Protocol 伺服器的全面集合。

## 安裝 (Installation)

```bash
# 使用 Copilot CLI
copilot plugin install swift-mcp-development@awesome-copilot
```

## 包含內容 (What's Included)

### 指令 (斜線指令) (Commands (Slash Commands))

| 指令 | 描述 |
|---------|-------------|
| `/swift-mcp-development:swift-mcp-server-generator` | 使用官方 MCP Swift SDK 套件產生完整的 Swift Model Context Protocol 伺服器專案。 |

### 代理程式 (Agents)

| 代理程式 | 描述 |
|-------|-------------|
| `swift-mcp-expert` | 使用現代化並行功能與官方 MCP Swift SDK 在 Swift 中建立 Model Context Protocol 伺服器的專家協助。 |

## 來源 (Source)

此外掛程式是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，這是一個社群驅動的 GitHub Copilot 擴充功能集合。
