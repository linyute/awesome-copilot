# Power Apps Code Apps 開發外掛程式 (Power Apps Code Apps Development Plugin)

Power Apps Code Apps 開發的完整工具包，包含專案骨架 (scaffolding)、開發標準，以及建立整合 Power Platform 之程式碼優先應用程式的專家指引。

## 安裝 (Installation)

```bash
# 使用 Copilot CLI
copilot plugin install power-apps-code-apps@awesome-copilot
```

## 包含內容 (What's Included)

### 指令 (斜線指令) (Commands (Slash Commands))

| 指令 | 描述 |
|---------|-------------|
| `/power-apps-code-apps:power-apps-code-app-scaffold` | 建立包含 PAC CLI 設定、SDK 整合與連接器組態的完整 Power Apps Code App 專案骨架 |

### 代理程式 (Agents)

| 代理程式 | 描述 |
|-------|-------------|
| `power-platform-expert` | 提供關於 Code Apps、畫布應用程式、Dataverse、連接器以及 Power Platform 最佳實務指引的 Power Platform 專家 |

## 來源 (Source)

此外掛程式是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，這是一個社群驅動的 GitHub Copilot 擴充功能集合。
