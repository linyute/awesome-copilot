# 前端網頁開發外掛程式 (Frontend Web Development Plugin)

現代前端網頁開發必備的提示、指引與聊天模式，包含 React、Angular、Vue、TypeScript 與 CSS 框架。

## 安裝 (Installation)

```bash
# 使用 Copilot CLI
copilot plugin install frontend-web-dev@awesome-copilot
```

## 包含內容 (What's Included)

### 指令 (斜線指令) (Commands (Slash Commands))

| 指令 | 描述 |
|---------|-------------|
| `/frontend-web-dev:playwright-explore-website` | 使用 Playwright MCP 進行測試用的網站探索 |
| `/frontend-web-dev:playwright-generate-test` | 使用 Playwright MCP 根據情境產生 Playwright 測試 |

### 代理程式 (Agents)

| 代理程式 | 描述 |
|-------|-------------|
| `expert-react-frontend-engineer` | 專家級 React 19.2 前端工程師，專精於現代化 hook、伺服器元件 (Server Components)、Actions、TypeScript 與效能最佳化 |
| `electron-angular-native` | 專為 Electron 應用程式量身打造的程式碼檢閱模式，包含 Node.js 後端 (main)、Angular 前端 (render) 以及原生整合層 (例如 AppleScript、shell 或原生工具)。此處不檢閱其他儲存庫中的服務。 |
