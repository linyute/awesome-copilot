# Power Platform MCP 連接器開發外掛程式 (Power Platform MCP Connector Development Plugin)

為 Microsoft Copilot Studio 開發整合 Model Context Protocol 之 Power Platform 自訂連接器的完整工具包

## 安裝 (Installation)

```bash
# 使用 Copilot CLI
copilot plugin install power-platform-mcp-connector-development@awesome-copilot
```

## 包含內容 (What's Included)

### 指令 (斜線指令) (Commands (Slash Commands))

| 指令 | 描述 |
|---------|-------------|
| `/power-platform-mcp-connector-development:power-platform-mcp-connector-suite` | 為 Copilot Studio 產生整合 MCP 的完整 Power Platform 自訂連接器 — 包含結構定義產生、疑難排解與驗證 |
| `/power-platform-mcp-connector-development:mcp-copilot-studio-server-generator` | 產生專為 Copilot Studio 整合最佳化的完整 MCP 伺服器實作，具備正確的結構定義約束與可串流的 HTTP 支援 |

### 代理程式 (Agents)

| 代理程式 | 描述 |
|-------|-------------|
| `power-platform-mcp-integration-expert` | 具備 MCP 整合之 Copilot Studio Power Platform 自訂連接器開發專家 — 擁有關於結構定義、協定與整合模式的全面知識 |
