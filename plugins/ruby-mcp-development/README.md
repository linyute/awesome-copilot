# Ruby MCP 伺服器開發外掛程式 (Ruby MCP Server Development Plugin)

使用官方 MCP Ruby SDK gem 並搭配 Rails 整合支援，在 Ruby 中建立 Model Context Protocol 伺服器的完整工具包。

## 安裝 (Installation)

```bash
# 使用 Copilot CLI
copilot plugin install ruby-mcp-development@awesome-copilot
```

## 包含內容 (What's Included)

### 指令 (斜線指令) (Commands (Slash Commands))

| 指令 | 描述 |
|---------|-------------|
| `/ruby-mcp-development:ruby-mcp-server-generator` | 使用官方 MCP Ruby SDK gem 產生完整的 Ruby Model Context Protocol 伺服器專案。 |

### 代理程式 (Agents)

| 代理程式 | 描述 |
|-------|-------------|
| `ruby-mcp-expert` | 使用官方 MCP Ruby SDK gem 與 Rails 整合在 Ruby 中建立 Model Context Protocol 伺服器的專家協助。 |

## 來源 (Source)

此外掛程式是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，這是一個社群驅動的 GitHub Copilot 擴充功能集合。
