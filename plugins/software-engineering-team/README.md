# 軟體工程團隊外掛程式 (Software Engineering Team Plugin)

7 個專用代理程式，涵蓋從 UX 設計與架構到安全性與 DevOps 的完整軟體開發生命週期。

## 安裝 (Installation)

```bash
# 使用 Copilot CLI
copilot plugin install software-engineering-team@awesome-copilot
```

## 包含內容 (What's Included)

### 代理程式 (Agents)

| 代理程式 | 描述 |
|-------|-------------|
| `se-ux-ui-designer` | 用於 Figma 與設計工作流程的待辦工作 (Jobs-to-be-Done) 分析、使用者旅程地圖與 UX 研究產出 |
| `se-technical-writer` | 用於建立開發人員文件、技術部落格、教學課程與教育內容的技術文件撰寫專家 |
| `se-gitops-ci-specialist` | 用於 CI/CD 管線、部署除錯與 GitOps 工作流程的 DevOps 專家，專注於使部署變得平淡且可靠 |
| `se-product-manager-advisor` | 用於建立 GitHub issue、將商業價值與使用者需求對齊，以及做出資料驅動產品決策的產品管理指引 |
| `se-responsible-ai-code` | 負責任 AI 專家，透過偏見預防、協助工具合規性、倫理開發與包容性設計，確保 AI 為每個人服務 |
| `se-system-architecture-reviewer` | 具備 Well-Architected 框架、設計驗證以及 AI 與分散式系統延展性分析能力的系統架構檢閱專家 |
| `se-security-reviewer` | 具備 OWASP Top 10、零信任、LLM 安全性與企業安全性標準的安全性導向程式碼檢閱專家 |

## 來源 (Source)

此外掛程式是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，這是一個社群驅動的 GitHub Copilot 擴充功能集合。
