# CAST Imaging Agent 外掛程式

專為軟體分析、影響評估、結構品質建議以及使用 CAST Imaging 進行架構檢閱而設計的專業 Agent 完整收藏。

## 安裝

```bash
# 使用 Copilot CLI
copilot plugin install cast-imaging@awesome-copilot
```

## 包含內容

### Agent

| Agent | 說明 |
|-------|-------------|
| `cast-imaging-software-discovery` | 專用 Agent，透過 CAST Imaging 的靜態程式碼分析，進行全面的軟體應用程式探索與架構映射。 |
| `cast-imaging-impact-analysis` | 專用 Agent，使用 CAST Imaging 進行全面的軟體系統變更影響評估與風險分析。 |
| `cast-imaging-structural-quality-advisor` | 專用 Agent，使用 CAST Imaging 識別、分析程式碼品質問題並提供修復指引。 |
