# TypeScript MCP 伺服器開發外掛程式 (TypeScript MCP Server Development Plugin)

使用官方 SDK 在 TypeScript/Node.js 中建立 Model Context Protocol (MCP) 伺服器的完整工具包。包含最佳實務指引、產生伺服器的提示以及提供指引的專家聊天模式。

## 安裝 (Installation)

```bash
# 使用 Copilot CLI
copilot plugin install typescript-mcp-development@awesome-copilot
```

## 包含內容 (What's Included)

### 指令 (斜線指令) (Commands (Slash Commands))

| 指令 | 描述 |
|---------|-------------|
| `/typescript-mcp-development:typescript-mcp-server-generator` | 建立包含工具、資源與正確組態的完整 TypeScript MCP 伺服器專案 |

### 代理程式 (Agents)

| 代理程式 | 描述 |
|-------|-------------|
| `typescript-mcp-expert` | 開發 TypeScript Model Context Protocol (MCP) 伺服器的專家助手 |

## 來源 (Source)

此外掛程式是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，這是一個社群驅動的 GitHub Copilot 擴充功能集合。
