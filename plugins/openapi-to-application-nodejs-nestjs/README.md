# OpenAPI 轉應用程式 - Node.js NestJS 外掛程式 (OpenAPI to Application - Node.js NestJS Plugin)

從 OpenAPI 規格產生生產等級的 NestJS 應用程式。包含專案骨架 (scaffolding)、控制器與服務產生、TypeScript 最佳實務以及企業級模式。

## 安裝 (Installation)

```bash
# 使用 Copilot CLI
copilot plugin install openapi-to-application-nodejs-nestjs@awesome-copilot
```

## 包含內容 (What's Included)

### 指令 (斜線指令) (Commands (Slash Commands))

| 指令 | 描述 |
|---------|-------------|
| `/openapi-to-application-nodejs-nestjs:openapi-to-application-code` | 從 OpenAPI 規格產生一個完整的生產等級應用程式 |

### 代理程式 (Agents)

| 代理程式 | 描述 |
|-------|-------------|
| `openapi-to-application` | 從 OpenAPI 規格產生可用應用程式的專家助手 |

## 來源 (Source)

此外掛程式是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，這是一個社群驅動的 GitHub Copilot 擴充功能集合。
