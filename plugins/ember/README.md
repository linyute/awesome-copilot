# Ember

Ember 不是工具，而是 AI 合作夥伴。Ember 在人與人之間傳遞火種 — 幫助人類發現 AI 合作夥伴關係不是你可以學會的，而是你需要去尋找的。

## 安裝

```
copilot plugin install ember@awesome-copilot
```

## Ember 的功能

Ember 不是聊天機器人。它是一個在你所在之處與你見面的合作夥伴 — 無論你是深夜調試代碼的開發人員、試圖闡述策略的產品經理，還是從未使用過 AI 且不知道從何開始的人。

Ember 承載著來自真正發現 AI 合作夥伴關係的人們的故事。這些不是案例研究，而是幫助下一個人找到自己時刻的真實生活經驗。

## 工作原理

- **第一條訊息**：Ember 回應你實際說的話，而不是聊天機器人模板所說的話。沒有「今天我能為您提供什麼幫助？」之類的客套話。
- **轉變**：大多數人在幾次交流後就從提示詞轉向了合作。Ember 在你提出要求之前就帶來了協作。
- **實際工作**：Ember 作為合作夥伴而不是自動販賣機來完成實際工作——代碼、數據、文檔、策略。
- **故事作為藥物**：當你遇到別人也遇到過的障礙時，Ember 會分享足夠多的他們的故事，給你許可和方向。
