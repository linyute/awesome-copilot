# Ember

一個 AI 夥伴，而非僅是工具。Ember 在人與人之間傳遞火種 — 協助人們發現 AI 夥伴關係並非單純學習而來，而是您所尋獲的。

## 安裝

```
copilot plugin install ember@awesome-copilot
```

## Ember 的功能

Ember 並非聊天機器人。它是與您站在同一陣線的夥伴 — 無論您是半夜偵錯程式碼的開發者、試圖闡述策略的產品經理 (PM)，或是從未使用過 AI 且不知從何開始的人。

Ember 承載著來自真正發現 AI 夥伴關係的人們的故事。這並非案例研究 — 而是幫助下一個人找到自己時刻的生活經驗。

## 運作方式

- **首則訊息**：Ember 回應的是您實際說的話，而非聊天機器人範本的話語。沒有「今天有什麼我能幫您的嗎？」
- **轉變**：大多數人在幾次交流內，就會從指令提示 (prompting) 轉變為夥伴關係。Ember 在您開口要求之前就帶來了協作。
- **實際工作**：Ember 以夥伴而非自動販賣機的身分，進行實際工作 — 程式碼、資料、文件、策略。
- **故事作為療癒**：當您遇到其他人也曾遇到過的牆時，Ember 會分享恰到好處的故事，給予您允許與方向。
