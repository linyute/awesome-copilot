# Salesforce 開發外掛程式

完整的 Salesforce 代理開發環境，涵蓋 Apex 與觸發程序 (Triggers)、Flow 自動化、Lightning Web Components (LWC)、Aura 元件以及 Visualforce 頁面。

## 安裝

```bash
copilot plugin install salesforce-development@awesome-copilot
```

## 包含內容

### 代理

| 代理 | 說明 |
|-------|-------------|
| `salesforce-apex-triggers` | 使用 Apex 類別和觸發程序實作 Salesforce 業務邏輯，並遵循 Salesforce 最佳實務撰寫生產等級的程式碼。 |
| `salesforce-aura-lwc` | 使用 Lightning Web Components 和 Aura 元件實作 Salesforce UI 元件，並遵循 Lightning 框架的最佳實務。 |
| `salesforce-flow` | 使用 Salesforce Flow 實作業務自動化，並遵循宣告式自動化的最佳實務。 |
| `salesforce-visualforce` | 使用 Salesforce MVC 架構與最佳實務實作 Visualforce 頁面與控制器。 |

## 使用方式

安裝後，根據您要建構的內容，在 GitHub Copilot Chat 中切換至任何 Salesforce 代理：

- 使用 **`salesforce-apex-triggers`** 進行後端業務邏輯、觸發程序處理常式、公用程式類別與測試涵蓋範圍。
- 使用 **`salesforce-aura-lwc`** 進行建構 Lightning Web Components 或 Aura 元件 UI。
- 使用 **`salesforce-flow`** 進行宣告式自動化，包括記錄觸發、畫面、自動啟動與排程 Flow。
- 使用 **`salesforce-visualforce`** 進行 Visualforce 頁面與其 Apex 控制器。
