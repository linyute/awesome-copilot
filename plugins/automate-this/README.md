# Automate This

你是否有些每週都要重複執行的動作 —— 那些需要點擊十五次、切換四個應用程式、複製貼上到試算表並寄出電子郵件，讓你恨不得把筆記型電腦丟進大海的流程？只要錄製一次你操作的過程，將影片交給 Copilot CLI，讓它為你撰寫自動化腳本。

## 運作方式

1. **錄製你的螢幕。** 使用 QuickTime、OBS、Loom 或任何你現有的工具。完全照你平時的操作方式執行流程。如果你想在錄製時進行講解（例如：「現在我正在下載這份報告，因為財務部門每週一都需要它」）那就更好了 —— 此外掛程式會逐字記錄你的旁白，並用它來理解你執行每個步驟的*原因*，而不僅僅是你點擊了*什麼*。

2. **將錄製檔案放在桌面**（或任何地方 —— 只需要一個檔案路徑）。

3. **告訴 Copilot CLI 進行分析：**
   ```
   copilot
   > /skills
   > (select automate-this)
   > Analyze ~/Desktop/weekly-report-process.mov and automate it
   ```

4. **檢視分析結果。** 此外掛程式每隔幾秒就會從你的影片中擷取畫面，逐字記錄任何旁白，並將你的流程重建為編號的步驟列表。它會在提出任何建議之前要求你確認內容是否正確。

5. **選擇你的自動化程度。** 你將獲得最多三種方案，範圍從「處理最繁瑣部分的一行指令」到「包含排程和錯誤處理的完整腳本」。每一種方案都使用你已經安裝的工具 —— 不會有令人意外的依賴鏈。

## 底層原理

此外掛程式使用 `ffmpeg` 從你的錄製中擷取畫面和音訊。如果你有旁白，它會使用 OpenAI 的 Whisper（在你本機執行，而非雲端）來逐字記錄你說的話。這些畫面會傳送到 AI 模型，它可以閱讀螢幕上的文字、識別應用程式、查看你的點擊行為，並遵循你的流程流程。

在提出自動化建議之前，此外掛程式會檢查你的環境 —— 安裝了什麼、使用什麼 shell、有哪些腳本語言可用 —— 因此它只會建議你實際上可以執行的方案，而不需要花一個小時進行設定。

## 先決條件

- **ffmpeg** (必備) — 用於從錄製檔案中擷取畫面與音訊
  ```bash
  brew install ffmpeg
  ```

- **Whisper** (選配) — 只有在你的錄製檔案有旁白時才需要
  ```bash
  pip install openai-whisper
  ```
  或是 C++ 版本：`brew install whisper-cpp`

如果未安裝 Whisper 且你的錄製中有音訊，此外掛程式會提醒你，並詢問是否僅以視覺分析繼續。

## 安裝

```bash
copilot plugin install automate-this@awesome-copilot
```

## 適合分析的內容

當你的錄製內容清楚呈現操作過程時，此外掛程式的效果最好。以下是一些適合自動化的流程範例：

- **報告產出** — 從儀表板下載資料、過濾、格式化，並寄送給其他人
- **檔案整理** — 將下載檔案分類到資料夾、按日期重新命名檔案、清理重複檔案
- **資料輸入** — 從一個應用程式複製資訊並輸入到另一個應用程式
- **開發環境設定** — 每次啟動新專案時需要執行的一系列指令和設定變更
- **重複的終端機工作流程** — 使用不同輸入執行相同的指令序列
- **基於電子郵件的工作流程** — 從電子郵件擷取資料、處理，並回覆結果

## 仍需手動的部分

此外掛程式會為機械化步驟提出自動化方案。它保留了人類的判斷 —— 如果你的錄製顯示你停下來檢查某些內容，或根據所見內容做出決定，該步驟在自動化中仍將保持手動，並會提示你輸入。

## 範例 (Example)

假設您錄製了自己執行每週任務的過程：您開啟瀏覽器，導覽至內部儀表板，下載三個 CSV 匯出檔案，在 Excel 中分別開啟，篩選標示為「待處理 (pending)」的資料列，將它們合併成一個工作表，並透過電子郵件傳送給您的經理。

該外掛程式將重建該程序，並與您確認，然後提出類似以下的建議：

- **第 1 層：** 一個 `curl` 指令，一次下載所有三個 CSV（如果儀表板有直接下載的 URL），完全跳過瀏覽器。
- **第 2 層：** 一個 Python 程式碼，下載 CSV、使用 pandas 篩選「待處理」資料列、合併它們並儲存結果 — 準備好作為電子郵件附件。
- **第 3 層：** 相同的程式碼，加上透過您的郵件用戶端自動傳送電子郵件，並於每週一上午 8 點透過 `launchd` 執行。

每一層都包含可執行的程式碼、使用試執行進行測試的說明，以及在發生錯誤時如何撤銷任何操作的指引。

## 隱私 (Privacy)

影格擷取 (ffmpeg) 和音訊逐字稿 (Whisper) 完全在您的電腦上執行。擷取的影格和音訊會寫入安全的暫存目錄，並在分析完成時刪除。擷取的影格和逐字稿會傳送到支援您 Copilot CLI 工作階段的 AI 模型進行分析 — 這與所有 Copilot 互動所使用的模型和資料管道相同。不會將任何資料傳送至 Copilot 已使用的服務之外的其他第三方服務。
