# Kotlin MCP 伺服器開發外掛程式 (Kotlin MCP Server Development Plugin)

使用官方 io.modelcontextprotocol:kotlin-sdk 函式庫在 Kotlin 中建立 Model Context Protocol (MCP) 伺服器的完整工具包。包含最佳實務指引、產生伺服器的提示以及提供指引的專家聊天模式。

## 安裝 (Installation)

```bash
# 使用 Copilot CLI
copilot plugin install kotlin-mcp-development@awesome-copilot
```

## 包含內容 (What's Included)

### 指令 (斜線指令) (Commands (Slash Commands))

| 指令 | 描述 |
|---------|-------------|
| `/kotlin-mcp-development:kotlin-mcp-server-generator` | 使用官方 io.modelcontextprotocol:kotlin-sdk 函式庫建立具備正確結構、相依性與實作的完整 Kotlin MCP 伺服器專案。 |

### 代理程式 (Agents)

| 代理程式 | 描述 |
|-------|-------------|
| `kotlin-mcp-expert` | 使用官方 SDK 在 Kotlin 中建立 Model Context Protocol (MCP) 伺服器的專家助手。 |

## 來源 (Source)

此外掛程式是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，這是一個社群驅動的 GitHub Copilot 擴充功能集合。
