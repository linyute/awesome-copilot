# Java MCP 伺服器開發外掛程式 (Java MCP Server Development Plugin)

使用官方 MCP Java SDK 並搭配回應式串流 (reactive streams) 與 Spring Boot 整合，在 Java 中建立 Model Context Protocol 伺服器的完整工具包。

## 安裝 (Installation)

```bash
# 使用 Copilot CLI
copilot plugin install java-mcp-development@awesome-copilot
```

## 包含內容 (What's Included)

### 指令 (斜線指令) (Commands (Slash Commands))

| 指令 | 描述 |
|---------|-------------|
| `/java-mcp-development:java-mcp-server-generator` | 使用官方 MCP Java SDK 並搭配回應式串流與選用的 Spring Boot 整合，產生完整的 Java Model Context Protocol 伺服器專案。 |

### 代理程式 (Agents)

| 代理程式 | 描述 |
|-------|-------------|
| `java-mcp-expert` | 使用回應式串流、官方 MCP Java SDK 與 Spring Boot 整合在 Java 中建立 Model Context Protocol 伺服器的專家協助。 |
