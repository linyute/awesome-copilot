# 安全性與程式碼品質外掛程式 (Security & Code Quality Plugin)

安全性框架、協助工具指引、效能最佳化與程式碼品質最佳實務，用於建立安全、可維護且高效能的應用程式。

## 安裝 (Installation)

```bash
# 使用 Copilot CLI
copilot plugin install security-best-practices@awesome-copilot
```

## 包含內容 (What's Included)

### 指令 (斜線指令) (Commands (Slash Commands))

| 指令 | 描述 |
|---------|-------------|
| `/security-best-practices:ai-prompt-engineering-safety-review` | 全面的 AI 提示工程安全性檢閱與改善提示。分析提示的安全性、偏見、安全性弱點與有效性，同時透過廣泛的框架、測試方法論與教育內容提供詳細的改善建議。 |

## 來源 (Source)

此外掛程式是 [Awesome Copilot](https://github.com/github/awesome-copilot) 的一部分，這是一個社群驅動的 GitHub Copilot 擴充功能集合。
