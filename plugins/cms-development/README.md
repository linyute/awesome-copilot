# CMS 開發外掛程式 (CMS Development Plugin)

用於內容管理系統開發的工具集，涵蓋佈景主題、外掛程式、管理員工具、媒體工作流程、Markdown 渲染以及靜態匯出管線。

## 安裝 (Installation)

```bash
# 使用 Copilot CLI
copilot plugin install cms-development@awesome-copilot
```

## 內含項目 (What's Included)

### 技能 (Skills)

| 技能 | 描述 |
|-------|-------------|
| `content-management-systems` | 跨平台（如 WordPress, Shopify, Drupal, Webflow 和 AEM）的 CMS 特定工作流程。涵蓋佈景主題、外掛程式、管理員面板、內容模型、媒體管線和靜態匯出。 |
| `markdown-to-html` | Markdown 渲染、轉換和 HTML 輸出工作流程。 |
| `quasi-coder` | 解讀網頁設計描述，並從速記、準程式碼 (quasi-code) 和自然語言描述中實作程式碼。 |
| `web-coder` | CMS 任務通常依賴的前端、瀏覽器和 HTTP 工作。 |

## 何時使用 (When to Use)

當工作中心為以下任何一項時，請安裝此外掛程式：

- 在任何 CMS 平台上開發或自訂佈景主題和範本
- 建立或修改外掛程式、應用程式、模組或擴充功能
- 處理管理員或編輯介面、內容表單或發佈流程
- 處理媒體上傳、資產管線或經撰寫的檔案儲存
- 實作或偵錯 Markdown 渲染和內容轉換
- 建立靜態匯出、部署或伺服器呈現的輸出管線
- 建立內容類型、分類 (taxonomy)、Slug、Metadata 或遷移結構描述 (migration schemas) 的模型
