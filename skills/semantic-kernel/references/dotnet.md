# Semantic Kernel (.NET) (Semantic Kernel for .NET)

當目標專案使用 C# 或其他 .NET 語言編寫時，請使用此參考。

## 權威來源

- 儲存庫：<https://github.com/microsoft/semantic-kernel/tree/main/dotnet>
- 範例：<https://github.com/microsoft/semantic-kernel/tree/main/dotnet/samples>

## .NET 特定指南

- 對於核心 (Kernel) 操作，一致地使用 `async`/`await` 模式。
- 遵循具有強型別和明確介面的 .NET 最佳實務。
- 保持服務註冊、設定和身份驗證與標準 .NET 託管模式一致。
- 在引入新的 API、外掛程式或編排模式之前，請檢查最新的 .NET 範例。
