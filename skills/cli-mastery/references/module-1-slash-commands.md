# 模組 1：斜線指令 (Module 1: Slash Commands)

一次教導一個類別，並提供範例和「何時使用」指引。

## 入門 (Getting Started)

| 指令 | 它的作用 | 何時使用 |
|---------|-------------|-------------|
| `/login` | 向 GitHub 進行驗證 | 首次啟動或工作階段過期 |
| `/logout` | 登出 | 切換帳號 |
| `/help` | 顯示所有指令 | 迷失方向時 |
| `/exit` `/quit` | 結束 CLI | 完成工作時 |
| `/init` | 引導產生 copilot-instructions.md | 新存放庫設定 |
| `/terminal-setup` | 設定多行輸入 | 首次設定 |

## 模型與 Agent (Models & Agents)

| 指令 | 它的作用 | 何時使用 |
|---------|-------------|-------------|
| `/model` | 切換 AI 模型 | 需要不同的功能/速度 |
| `/agent` | 瀏覽/選擇 Agent | 委派給專家 |
| `/fleet` | 啟用平行子 Agent | 複雜的多部分任務 |
| `/tasks` | 檢視背景任務 | 檢查執行中的子 Agent |

## 程式碼與檢閱 (Code & Review)

| 指令 | 它的作用 | 何時使用 |
|---------|-------------|-------------|
| `/diff` | 檢閱目前目錄中的變更 | 提交前 |
| `/review` | 執行程式碼檢閱 Agent | 獲取變更的回饋 |
| `/lsp` | 管理語言伺服器 | 需要跳至定義、診斷 |
| `/ide` | 連接至 IDE 工作區 | 想要 IDE 整合 |

## 工作階段與上下文 (Session & Context)

| 指令 | 它的作用 | 何時使用 |
|---------|-------------|-------------|
| `/context` | 顯示權杖使用量視覺化 | 上下文變得龐大時 |
| `/usage` | 顯示工作階段指標 | 檢查 Premium 請求計數 |
| `/compact` | 壓縮對話歷史記錄 | 接近上下文限制時 |
| `/session` | 顯示工作階段資訊 | 需要工作階段詳細資訊 |
| `/resume` | 切換至不同的工作階段 | 繼續之前的工作 |
| `/rename` | 重新命名目前工作階段 | 更好的組織 |
| `/share` | 將工作階段匯出為 Markdown/Gist | 與團隊分享 |
| `/copy` | 將最後一個回應複製到剪貼簿 | 快速抓取 AI 輸出 |
| `/clear` | 清除對話歷史記錄 | 重新開始 |

## 權限與目錄 (Permissions & Directories)

| 指令 | 它的作用 | 何時使用 |
|---------|-------------|-------------|
| `/allow-all` | 啟用所有權限 | 信任的環境，快速行動 |
| `/add-dir` | 加入信任的目錄 | 跨專案工作 |
| `/list-dirs` | 顯示允許的目錄 | 檢查存取範圍 |
| `/cwd` | 變更工作目錄 | 切換專案上下文 |
| `/reset-allowed-tools` | 撤銷工具核准 | 加強安全性 |

## 設定與自訂 (Configuration & Customization)

| 指令 | 它的作用 | 何時使用 |
|---------|-------------|-------------|
| `/instructions` | 檢視作用中的指令檔案 | 偵錯自訂行為 |
| `/experimental` | 切換實驗性功能 | 嘗試自動駕駛模式 |
| `/theme` | 變更終端機佈景主題 | 個人化 |
| `/streamer-mode` | 隱藏敏感資訊 | 直播/演示 |
| `/changelog` | 顯示版本資訊 | 更新後 |
| `/update` | 更新 CLI | 有新版本可用時 |
| `/feedback` | 提交回饋 | 報告錯誤或提出請求 |
