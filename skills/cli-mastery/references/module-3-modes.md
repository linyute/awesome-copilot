# 模組 3：互動模式 (Module 3: Interaction Modes)

## 互動模式 (Interactive Mode，預設)

- AI 針對您的提示立即採取行動
- 針對高風險操作會詢問許可
- 適用於：快速任務、偵錯、探索程式碼
- 80% 的時間您將待在這裡

## 計畫模式 (Plan Mode，使用 `Shift+Tab` 或 `/plan`)

- AI 首先建立逐步計畫
- 您在執行前檢閱並核准
- 適用於：複雜重構、架構變更、高風險操作
- 核心見解：當錯誤代價高昂時請使用此模式

## 自動駕駛模式 (Autopilot Mode，實驗性，`/experimental`)

- AI 在不詢問確認的情況下採取行動
- 適用於：信任的環境、長時間執行的任務
- 請謹慎使用 — 搭配 `/allow-all` 或 `--yolo`

## 模式比較 (Mode Comparison)

| 功能 | 互動模式 | 計畫模式 | 自動駕駛模式 |
|---------|------------|------|-----------|
| 速度 | 快 | 較慢 | 最快 |
| 安全性 | 中 | 最高 | 最低 |
| 控制 | 您核准每個動作 | 您核准計畫 | AI 完全自主 |
| 最適用於 | 日常任務 | 複雜變更 | 重複/信任的工作 |
| 切換方式 | 預設 | `Shift+Tab` 或 `/plan` | `/experimental` (啟用)，然後按 `Shift+Tab` |

教學重點：在正確的時間選擇正確的模式 = 10 倍生產力。
