# 遊戲發行 (Game Publishing)

此參考指南涵蓋了發行網頁遊戲的三大支柱：分發管道與平台、推廣策略以及營利模式。

## 遊戲分發 (Game Distribution)

遊戲分發涵蓋了玩家發現並存取您遊戲的管道與平台。選擇正確的分發策略取決於您的目標對象、遊戲類型以及商業目標。

### 自我代管 (Self-Hosting)

自我代管可讓您對遊戲擁有最大的控制權，並能夠立即推送更新而無須等待應用程式商店審核。

- 將遊戲上傳到遠端伺服器，並使用一個好記且響亮的網域名稱。
- 串接並縮減 (minify) 原始碼以減少傳輸量。
- 混淆 (uglify) 程式碼以增加逆向工程的難度，保護智慧財產權。
- 如果您計畫為 iTunes 或 Steam 等封閉商店封裝遊戲，請提供線上展示版本。
- 考慮在 GitHub Pages 上代管，以獲得免費代管、版本控制以及潛在的社群貢獻。

### 發行商與入口網站 (Publishers and Portals)

獨立遊戲入口網站提供來自高流量網站的自然推廣，以及透過廣告或收入拆帳實現營利的潛力。

**熱門的獨立入口網站：**

- HTML5Games.com
- GameArter.com
- MarketJS.com
- GameFlare
- GameDistribution.com
- GameSaturn.com
- Playmox.com
- Poki (developers.poki.com)
- CrazyGames (developer.crazygames.com)

**授權選項：**

- 獨家授權：限制僅由單一買家分發，以換取較高的單筆交易收入。
- 非獨家授權：在多個入口網站廣泛分發，以獲得更廣泛的觸及。

### 網頁商店 (Web Stores)

**Chrome 線上應用程式商店：**

- 需要一個資訊清單 (manifest) 檔案和一個包含遊戲資源的壓縮套件。
- 僅需對遊戲進行極少量的修改。
- 簡單的線上提交表單。

### 原生行動商店 (Native Mobile Stores)

**iOS App Store：**

- 要求嚴格，且有 1-2 週的審核等待期。
- 競爭極其激烈，擁有數十萬個應用程式。
- 通常較偏好付費遊戲。
- 最著名的行動商店，但最難脫穎而出。

**Google Play (Android)：**

- 要求比 iOS 寬鬆。
- 每日提交量龐大。
- 偏好 Freemium 模式（免費下載搭配應用程式內購買或廣告）。
- 大多數付費 iOS 遊戲在 Android 上以免費遊玩的形式出現。

**其他行動平台 (Windows Phone, BlackBerry 等)：**

- 競爭較少，較容易獲得曝光。
- 市佔率較小但較不擁擠。

### 原生桌面端 (Native Desktop)

**Steam：**

- 最大的桌面遊戲分發平台。
- 透過 Steam Direct 計畫供獨立開發者加入。
- 需要支援多個平台（Windows、macOS、Linux）並分別上傳。
- 必須處理跨平台相容性問題。

**Humble Bundle：**

- 主要是曝光與推廣機會。
- 以低價捆綁銷售的定價模式。
- 比起產生直接收入，更注重獲得曝光率。

### 封裝工具 (Packaging Tools)

將 HTML5 遊戲分發到封閉生態系統的工具：

| 工具 | 平台 |
|------|-----------|
| Ejecta | iOS (ImpactJS 專用) |
| NW.js | Windows, Mac, Linux |
| Electron | Windows, Mac, Linux |
| Intel XDK | 多個平台 |
| Manifold.js | iOS, Android, Windows |

### 平台策略 (Platform Strategy)

- **行動優先：** 行動裝置佔了 HTML5 遊戲流量的絕大部分。設計持握裝置時僅需一兩根手指即可遊玩的遊戲。
- **桌面端用於開發：** 在行動端偵錯前，先在桌面端建構並測試。
- **多平台：** 即使主要針對行動端，也應支援桌面端。HTML5 遊戲具備「一次編寫，處處部署」的優勢。
- **多元化：** 不要依賴單一商店。分佈在多個平台上以降低風險。
- **即時更新：** 網頁分發的關鍵優勢之一是能夠快速推送錯誤修正，而無須等待應用程式商店審核。

## 遊戲推廣 (Game Promotion)

遊戲推廣需要持續的多管道策略。大多數推廣方法是免費的，這使得預算有限的獨立開發者也能負擔。知名度與遊戲品質同樣重要 — 即使是優秀的遊戲，若沒有推廣也會失敗。

### 網站與部落格 (Website and Blog)

**必備的網站組件：**

- 螢幕截圖與遊戲預告片。
- 詳細說明與可下載的媒體素材包 (press kits)（使用 Presskit() 等工具）。
- 系統需求與可用平台。
- 支援與聯絡資訊。
- 可遊玩的展示版 (demo)，至少是基於瀏覽器的版本。
- 針對搜尋引擎最佳化 (SEO) 以提高可發現性。

**部落格策略：**

- 記錄開發過程、遇到的錯誤以及學到的教訓。
- 發佈每月進度報告。
- 持續的內容創作會隨時間提高 SEO 排名。
- 建立公信力與社群聲望。

### 社群媒體 (Social Media)

- 在 Twitter/X 等平台上使用 `#gamedev` 標籤進行社群互動。
- 保持真實，避免生硬的廣告或枯燥的新聞稿。
- 分享開發技巧、行業見解以及幕後花絮。
- 監控可能報導您遊戲的 YouTube 和 Twitch 實況主。
- 參與 HTML5GameDevs.com 等論壇。
- 真誠地與社群互動。回答問題、給予支持，並避免不斷的自我推銷。
- 提供折扣與競賽獎品以建立友好關係。

### 媒體聯繫 (Press Outreach)

- 研究專門報導您遊戲類型與平台的媒體機構。
- 聯繫記者與評論員時要謙虛、有禮且有耐心。
- 避免大量發送無關的提交。精確鎖定您的聯繫對象。
- 高品質的遊戲搭配誠實的做法，成功率最高。
- 參考 Pixel Prospector 的「如何聯繫媒體」等指南以獲取詳細策略。

### 競賽 (Competitions)

- 參加遊戲開發競賽 (game jams) 以建立人脈並獲得社群曝光。
- 強制性的主題會激發創意並迫使創新。
- 獲勝會帶來主辦方的自動推廣以及社群關注。
- 非常適合發佈早期展示版並建立聲望。

### 教學與教育內容 (Tutorials and Educational Content)

- 記錄並教導您在遊戲中實作的內容。
- 在文章和教學中將您的遊戲作為實際案例研究。
- 在 Tuts+ Game Development 等平台發佈，這些平台通常會支付稿費。
- 深入探討單一面向，並為讀者提供真正的價值。
- 雙重效益：既推廣了您的遊戲，又確立了您作為資深開發者的地位。

### 活動 (Events)

**研討會：**

- 針對開發期間克服的挑戰進行技術演講。
- 以您的遊戲作為真實範例演示 API 實作。
- 著重知識分享而非行銷。開發者對強硬的推銷手段特別反感。

**展覽與博覽會：**

- 在其他開發者中爭取攤位，與粉絲直接互動。
- 以獨特、原創的呈現方式脫穎而出。
- 提供真實的使用者測試與即時回饋。
- 有助於發現玩家有機發現的錯誤與問題。

### 促銷代碼 (Promo Codes)

- 建立分發免費或限量存取促銷代碼的能力。
- 分發給媒體、YouTube 和 Twitch 名人、競賽獲勝者以及社群影響力人士。
- 讓對的人獲得免費存取權限，可以產生針對數千名潛在玩家的免費廣告。

### 社群經營 (Community Building)

- 向您的受眾定期發送包含更新資訊的週報。
- 組織與您的遊戲或遊戲開發相關的線上競賽。
- 舉辦在地聚會，進行面對面的開發者交流。
- 展現熱情並建立信任與可靠性。
- 當您需要啟動支援或為發佈造勢時，您的社群會成為您的擁護者。

### 關鍵推廣原則 (Key Promotion Principles)

| 因素 | 重要性 |
|--------|-----------|
| 一致性 | 所有管道的定期內容與參與 |
| 真實性 | 真誠的社群互動，而非交易式互動 |
| 耐心 | 建立關係與聲望需要時間 |
| 價值優先 | 在要求任何回報前提供值得消費的內容 |
| 多重管道 | 絕不依賴單一推廣策略 |

## 遊戲營利 (Game Monetization)

營利策略應與您的遊戲類型、目標受眾以及分發平台保持一致。多元化的收入來源可提供更好的業務穩定性。

### 付費遊戲 (Paid Games)

**模式：** 在玩家獲得存取權限前收取的固定預付價格。

- 需要大量的行銷投入以推動購買。
- 價格隨市場與品質而異：iOS 街機類約 $0.99 美元，Steam 上的桌面端 RPG 約 $20 美元。
- 成功取決於遊戲品質、市場研究以及行銷有效性。
- 研究市場趨勢並從失敗中快速學習。

### 應用程式內購買 (IAPs) (In-App Purchases (IAPs))

**模式：** 免費獲取遊戲，但提供付費的選用內容與功能。

**可購買內容的類型：**

- 獎勵關卡
- 更好的武器或法術
- 能量補充
- 遊戲內貨幣
- 進階功能與虛擬商品

**關鍵指標與考量：**

- 需要數千次下載才能產生有意義的收入。
- 通常每 1,000 名玩家中僅約 1 人會進行購買。
- 收益高度依賴推廣活動與玩家數量。
- 玩家數量是成功的關鍵因素。

### Freemium (Freemium)

**模式：** 免費遊戲，具備選用的進階功能與付費福利。

- 為遊戲增加價值，而非將核心內容限制在付費牆後。
- 避免玩家厭惡的「付費贏」(pay-to-win) 機制，那會損害留存率與聲望。
- 不要對遊戲進度設定付費牆。
- 專注於先提供愉快的免費體驗，然後提供進階增強功能。

**附加元件與 DLC：**

- 包含新角色、武器與故事內容的新關卡集。
- 需要一個具備既有知名度的穩定基礎遊戲。
- 為現有的、參與度高的玩家提供額外價值。

### 廣告 (Advertisements)

**模式：** 透過廣告顯示獲取被動收入，開發者與廣告網路共享收入。

**廣告網路：**

- **Google AdSense：** 最有效但未針對遊戲最佳化。對遊戲相關帳戶可能存在風險。
- **LeadBolt：** 針對遊戲的替代方案，實作較容易。
- **影片廣告：** 在載入畫期間顯示的片頭廣告 (Pre-roll) 格式正日益普及。

**投放策略：**

- 在遊戲工作階段之間或遊戲結束畫面上顯示廣告。
- 在廣告可見度與玩家體驗之間取得平衡。
- 保持廣告隱蔽，以避免惹惱玩家並損害留存率。
- 對於低流量遊戲，收入通常非常微薄。

**收入拆帳：** 與發行商通常採 70/30 或 50/50 拆帳。

### 授權 (Licensing)

**模式：** 支付一次性費用以獲取分發權。由發行商處理營利。

**獨家授權：**

- 僅出售給單一發行商。
- 交易後不能再以任何形式出售。
- 價格範圍：$2,000 到 $5,000 美元。
- 僅在利潤足以抵銷獨家性時才追求。出售後停止推廣該遊戲。

**非獨家授權：**

- 可同時出售給多個發行商。
- 發行商僅能在其自身的入口網站分發（網站鎖定）。
- 價格範圍：每家發行商約 $500 美元。
- 最受歡迎的授權方式。適合與多家發行商持續合作。

**訂閱模式：**

- 每款遊戲每月的被動收入。
- 價格範圍：每款遊戲每月 $20 到 $50 美元。
- 彈性的支付選項：整筆支付或按月支付。
- 風險：發行商可隨時取消。

**廣告收入拆帳：**

- 發行商帶動流量並拆分收益。
- 拆帳：70/30 或 50/50 交易，按月結算。
- 警告：新興或低品質的發行商可能總共僅提供低至 $2 美元。

**重要授權筆記：**

- 發行商可能要求實作自訂 API（將開發成本計入您的定價中）。
- 寧可接受知名、有信譽發行商較低的授權費，也不要冒險與不明買家交易。
- 透過網站或 HTML5 Gamedevs 論壇聯繫發行商。

### 品牌化與客製化工作 (Branding and Custom Work)

**具備品牌化的非獨家授權：**

- 客戶購買程式碼權限並實作其自身的圖形。
- 範例：將遊戲中的食物換成客戶品牌的產品。

**自由案接案品牌化：**

- 開發者重用現有的遊戲程式碼並增加客戶提供的圖形。
- 客戶主導實作細節。
- 價格隨品牌、客戶期望與工作範圍而有巨大差異。

### 其他營利策略 (Other Monetization Strategies)

**銷售數位資產：**

- 在 Envato Market 和 ThemeForest 等平台上銷售遊戲圖形與美術資產。
- 最適合能建立可重用資產的圖形設計師。
- 提供被動、微薄但穩定的收入。

**撰寫文章與教學：**

- 在 Tuts+ Game Development 等平台發佈遊戲開發文章，這些平台會支付稿費。
- 雙重效益：既推廣了您的遊戲，又產生了直接收入。
- 專注於真誠的知識分享，將您的遊戲作為實際範例。

**周邊商品 (Merchandise)：**

- T 恤、貼紙與品牌小物。
- 對於高度知名、視覺辨識度高的遊戲（例如憤怒鳥）最有利可圖。
- 某些開發者從周邊商品獲得的收入超過遊戲本身。
- 最適合作為多元化的次要收入來源。

**社群捐贈：**

- 在遊戲頁面增加捐贈按鈕。
- 有效性取決於社群關係的強度。
- 在玩家親自認識您並了解捐贈如何協助持續開發時效果最好。

### 營利摘要 (Monetization Summary)

| 模式 | 收入類型 | 最適合 | 風險等級 |
|-------|-------------|----------|------------|
| 付費遊戲 | 一次性 | 具備強大行銷的高品質遊戲 | 高 |
| 應用程式內購買 | 每筆交易 | 具備高下載量的熱門遊戲 | 中 |
| 廣告 | 被動/CPM | 休閒、具成癮性的遊戲 | 低-中 |
| 非獨家授權 | 一次性 (~$500) | 所有遊戲類型 | 低 |
| 獨家授權 | 一次性 ($2K-$5K) | 經過驗證的高品質遊戲 | 中 |
| 訂閱 | 每月被動 | 具有穩定記錄的遊戲 | 中 |
| 周邊商品 | 每筆銷售 | 具有視覺辨識度的熱門系列 | 高 |
| 文章/教學 | 每篇發佈 | 具有利基專業知識的開發者 | 低 |
