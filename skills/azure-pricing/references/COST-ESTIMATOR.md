# 成本估算器參考 (Cost Estimator Reference)

用於將 Azure 單位價格轉換為每月與年度成本估算值的公式與模式。

## 標準時間型運算

### 每月小時數

Azure 使用 **每月 730 小時** 作為標準計費週期 (365 天 × 24 小時 / 12 個月)。

```
每月成本 = 每小時單位價格 × 730
年度成本 = 每月成本 × 12
```

### 常見乘數

| 週期 | 小時數 | 運算方式 |
|--------|-------|-------------|
| 1 小時 | 1 | 單位價格 |
| 1 天 | 24 | 單位價格 × 24 |
| 1 週 | 168 | 單位價格 × 168 |
| 1 個月 | 730 | 單位價格 × 730 |
| 1 年 | 8,760 | 單位價格 × 8,760 |

## 服務特定公式

### 虛擬機器 (運算) (Virtual Machines - Compute)

```
每月成本 = 每小時價格 × 730
```

對於僅在上班時間執行的 VM（每天 8 小時，每月 22 天）：
```
每月成本 = 每小時價格 × 176
```

### Azure Functions

```
執行成本 = 每次執行價格 × 執行次數
運算成本 = 每 GB-s 價格 × (以 GB 為單位的記憶體 × 以秒為單位的執行時間 × 執行次數)
每月總計 = 執行成本 + 運算成本
```

免費補助：每月 100 萬次執行與 400,000 GB-s。

### Azure Blob 儲存 (Azure Blob Storage)

```
儲存成本 = 每 GB 價格 × 以 GB 為單位的儲存量
交易成本 = 每 10,000 次操作的價格 × (操作次數 / 10,000)
輸出 (Egress) 成本 = 每 GB 價格 × 以 GB 為單位的輸出量
每月總計 = 儲存成本 + 交易成本 + 輸出成本
```

### Azure Cosmos DB

#### 預留輸送量 (Provisioned Throughput)
```
每月成本 = (RU/s / 100) × 每 100 RU/s 價格 × 730
```

#### 無伺服器 (Serverless)
```
每月成本 = (總共消耗的 RU / 1,000,000) × 每 100 萬 RU 價格
```

### Azure SQL 資料庫 (Azure SQL Database)

#### DTU 模型
```
每月成本 = 每 DTU 價格 × DTU 數量 × 730
```

#### vCore 模型
```
每月成本 = vCore 價格 × vCore 數量 × 730 + 每 GB 儲存價格 × 儲存量 (GB)
```

### Azure Kubernetes 服務 (AKS)

```
每月成本 = 節點 VM 價格 × 730 × 節點數量
```

標準層的控制平面是免費的。

### Azure 應用程式服務 (Azure App Service)

```
每月成本 = 方案價格 × 730 (針對以小時計價的方案)
```

或者針對固定層級方案的固定每月價格。

### Azure OpenAI

```
每月成本 = (輸入權杖 (Tokens) 數量 / 1000) × 每 1K 權杖輸入價格
         + (輸出權杖 (Tokens) 數量 / 1000) × 每 1K 權杖輸出價格
```

## 預留 vs. 隨用隨付比較 (Reservation vs. Pay-As-You-Go Comparison)

呈現定價選項時，務必顯示比較：

```
| 定價模型 | 每月成本 | 年度成本 | 相較於 PAYG 的節省金額 |
|---------------|-------------|-------------|------------------|
| 隨用隨付 (PAYG) | $X | $Y | — |
| 1 年預留 | $A | $B | Z% |
| 3 年預留 | $C | $D | W% |
| 節省方案 (1 年) | $E | $F | V% |
| 節省方案 (3 年) | $G | $H | U% |
| 現貨 (Spot) (若可用) | $I | 不適用 (N/A) | T% |
```

節省百分比公式：
```
節省 % = ((PAYG 價格 - 預留價格) / PAYG 價格) × 100
```

## 成本摘要表範本

務必以此格式呈現結果：

```markdown
| 服務 | SKU | 區域 | 單位價格 | 單位 | 每月預估 | 年度預估 |
|---------|-----|--------|-----------|------|-------------|-------------|
| 虛擬機器 | Standard_D4s_v5 | 美國東部 | $0.192/hr | 1 小時 | $140.16 | $1,681.92 |
```

## 提示 (Tips)

- 在估算前務必先釐清**使用模式**（24/7 全天候執行 vs. 上班時間執行 vs. 偶發性執行）。
- 針對**儲存**，請詢問預期的資料量與存取模式。
- 針對**資料庫**，請詢問輸送量需求（RU/s、DTU 或 vCore）。
- 針對**無伺服器**服務，請詢問預期的呼叫次數與持續時間。
- 顯示時請四捨五入至小數點後第 2 位。
- 請注意，除非另有說明，否則價格皆以 **USD** 為單位。
