# Azure 服務對應

## 每個模式常用的 Azure 服務

- **訊息佇列**：Azure Service Bus, Azure Storage Queue, Event Hubs
- **快取**：Azure Cache for Redis, Azure Front Door 快取
- **API 閘道**：Azure API Management, Azure Application Gateway
- **身分識別**：Azure AD, Azure AD B2C
- **設定**：Azure App Configuration, Azure Key Vault
- **儲存**：Azure Storage (Blob, Table, Queue), Azure Cosmos DB
- **運算**：Azure Functions, Azure Container Apps, Azure Kubernetes Service
- **事件串流**：Azure Event Hubs, Azure Event Grid
- **CDN**：Azure CDN, Azure Front Door
