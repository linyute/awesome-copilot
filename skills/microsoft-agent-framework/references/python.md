# Microsoft 代理程式框架 (Python) (Microsoft Agent Framework for Python)

當目標專案使用 Python 編寫時，請使用此參考。

## 權威來源

- 儲存庫：<https://github.com/microsoft/agent-framework/tree/main/python>
- 範例：<https://github.com/microsoft/agent-framework/tree/main/python/samples>

## 安裝

對於新專案，使用以下命令安裝套件：

```bash
pip install agent-framework
```

## Python 特定指南

- 在整個代理程式和工作流操作中，使用現代化的非同步模式。
- 加入型別提示，即使在動態程式碼中也保持 API 的明確性。
- 在相依項和工具方面，遵循標準 Python 套件打包和環境實務。
- 以符合 Python 應用程式結構的方式使用中介軟體、上下文提供者和編排 (Orchestration) 模式。
- 在引入新的 API 或工作流模式之前，請檢查最新的 Python 範例。
